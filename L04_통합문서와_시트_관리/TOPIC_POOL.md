# 🎯 화두 풀 — 통합 문서와 시트 관리

> 아래 주제를 골라 직접 실습해보세요!  
> 여러 시트를 활용하고 다양한 파일 형식으로 저장합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 이력서/자기소개서 양식 ⭐ 추천
> 3개 시트로 구성된 이력서를 만들어보세요.
- 시트1: 인적사항 (이름, 연락처, 주소 등)
- 시트2: 학력 및 자격증
- 시트3: 자기소개서 (넓은 셀 병합 활용)
- 각 시트 탭에 다른 색상 적용
- PDF로 저장하여 인쇄 상태 확인

### 2. 월별 일기장
> 12개 시트(1월~12월)를 만들고 일기장 양식을 만들어보세요.
- 각 시트 이름: 1월, 2월, ... 12월
- 양식: A열(날짜), B열(날씨), C열(오늘의 기분), D열(일기 내용)
- 한 시트를 만든 뒤 Ctrl+드래그로 복사하면 빠릅니다!

### 3. 과목별 성적 관리
> 과목별로 시트를 나누어 성적을 관리해보세요.
- 시트: 국어, 수학, 영어, 과학, 사회
- 각 시트에 시험일, 단원, 점수를 기록

---

## 🟡 도전 (조금 더 생각 필요)

### 4. 여행 계획 노트
> 여행 계획을 체계적으로 정리해보세요.
- 시트1 "일정": 날짜별 시간대 일정
- 시트2 "예산": 항목별 예상 비용
- 시트3 "준비물": 준비물 체크리스트
- 시트4 "맛집": 가고 싶은 식당 목록
- 각 시트를 적절한 탭 색상으로 구분

### 5. CSV 파일 활용 실험
> 엑셀 데이터를 CSV로 저장하고 다양하게 활용해보세요.
- 간단한 주소록 표를 만들기
- xlsx로 저장 → CSV로 저장
- CSV 파일을 메모장으로 열어 내용 확인
- 다시 엑셀로 열어보기 — 서식이 유지되나요?

---

## 🔴 심화 (탐구해보기)

### 6. 동아리 관리 시스템
> 동아리(또는 모임) 관리를 위한 종합 문서를 만들어보세요.
- 시트1 "회원명단": 이름, 학년, 연락처, 가입일
- 시트2 "회비내역": 날짜, 이름, 납부금액, 잔액
- 시트3 "활동일지": 날짜, 활동내용, 참석자
- 시트4 "통계": (나중에 수식을 배우면 여기에 요약을 만들 예정!)
- 각 시트 양식을 만들고 시트 탭 색상, 틀 고정 적용
