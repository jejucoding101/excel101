# 📌 7강: 완벽한 인쇄 문서 만들기

> **핵심 포인트**: 화면에서 만든 표를 종이에 완벽하게 출력하기 위한 인쇄 영역, 페이지 설정, 머리글/바닥글을 배웁니다.

---

## 📖 이론 (20분)

### 왜 인쇄 설정이 중요한가?

엑셀에서 만든 표를 그냥 인쇄하면 이런 문제가 생깁니다:

```
❌ 흔한 인쇄 실수들:
• 표가 잘려서 두 페이지로 나뉨
• 2페이지부터 제목 행이 없어서 뭐가 뭔지 모름
• 여백이 너무 커서 내용이 작게 인쇄됨
• 가로로 긴 표인데 세로로 인쇄됨
```

### 인쇄 미리보기 (`Ctrl+P`)

인쇄 전에 **반드시** 미리보기로 확인하세요!

- `Ctrl+P` 또는 파일 → 인쇄
- 왼쪽: 인쇄 설정 옵션
- 오른쪽: 실제 인쇄될 모습 미리보기
- 하단: 페이지 번호 (여러 페이지일 때 화살표로 이동)

### 페이지 방향

| 방향 | 적합한 경우 |
|------|-----------|
| **세로(Portrait)** | 행이 많고 열이 적은 표 (목록형) |
| **가로(Landscape)** | 열이 많은 표 (넓은 표) |

설정: 페이지 레이아웃 → 방향 → 세로/가로

### 용지 크기

| 크기 | 규격 | 용도 |
|------|------|------|
| **A4** | 210×297mm | 가장 일반적 (기본값) |
| **B4** | 257×364mm | 큰 표 |
| **Letter** | 216×279mm | 미국 표준 |

설정: 페이지 레이아웃 → 크기

### 여백 (Margin)

용지 가장자리부터 내용까지의 빈 공간입니다.

```
┌──────────────────────────────┐
│ ↕ 위쪽 여백                   │
│  ┌────────────────────────┐  │
│  │                        │  │
│←→│      내용 영역          │←→│
│좌 │                        │우 │
│  └────────────────────────┘  │
│ ↕ 아래쪽 여백                  │
└──────────────────────────────┘
```

| 프리셋 | 위/아래 | 좌/우 | 용도 |
|--------|--------|-------|------|
| 보통 | 1.91cm | 1.78cm | 기본값 |
| 좁게 | 1.27cm | 0.64cm | 내용을 더 많이 |
| 넓게 | 2.54cm | 2.54cm | 여유 있게 |

> 💡 **팁**: 표가 한 페이지에 아슬아슬하게 안 들어갈 때, **"좁게"**로 바꾸면 해결되는 경우가 많습니다!

### 인쇄 영역 설정

시트 전체가 아니라 **특정 범위만** 인쇄하고 싶을 때:

1. 인쇄할 범위 선택
2. 페이지 레이아웃 → **인쇄 영역 설정**
3. `Ctrl+P`로 확인 → 선택한 범위만 인쇄됨!

해제: 페이지 레이아웃 → **인쇄 영역 지우기**

### 반복할 행/열 (인쇄 제목) ⭐

여러 페이지로 인쇄할 때, **모든 페이지에 제목 행을 반복** 표시합니다.

```
페이지 1:                    페이지 2:
┌──────────────────┐        ┌──────────────────┐
│ 이름  나이  점수  │ 제목행  │ 이름  나이  점수  │ ← 같은 제목!
├──────────────────┤        ├──────────────────┤
│ 1번   20   85    │        │ 51번  22   88    │
│ 2번   21   90    │        │ 52번  23   91    │
│ ...              │        │ ...              │
│ 50번  25   78    │        │ 100번 24   95    │
└──────────────────┘        └──────────────────┘
```

**설정 방법**:
1. 페이지 레이아웃 → **인쇄 제목**
2. "반복할 행" → 제목 행 클릭 (예: `$1:$1`)
3. 확인 → 모든 페이지에 1행이 반복됨!

### 머리글/바닥글

각 페이지 위/아래에 표시되는 정보입니다.

```
┌──────────────────────────────┐
│ 보고서 제목     작성자: 홍길동  │ ← 머리글 (Header)
├──────────────────────────────┤
│                              │
│         본문 내용             │
│                              │
├──────────────────────────────┤
│ 2025-03-15        1 / 3 페이지│ ← 바닥글 (Footer)
└──────────────────────────────┘
```

**설정 방법**:
- 삽입 → 머리글/바닥글 (또는 보기 → 페이지 레이아웃 보기)
- 자주 쓰는 요소: 페이지 번호, 날짜, 파일명, 시트명

### 페이지 나누기

긴 표를 원하는 위치에서 페이지를 나눌 수 있습니다.

- **보기 → 페이지 나누기 미리 보기** → 파란 점선을 드래그하여 위치 조절
- 또는: 나누고 싶은 행 선택 → 페이지 레이아웃 → 나누기 → 페이지 나누기 삽입

### 한 페이지에 맞추기

표를 강제로 한 페이지에 축소 인쇄하는 방법:

페이지 레이아웃 → 크기 조정:
- 너비: 1페이지
- 높이: 1페이지 (또는 "자동")

> ⚠️ **주의**: 너무 많은 내용을 한 페이지에 맞추면 글씨가 매우 작아집니다!

### ⌨️ 이번 강의 필수 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+P` | 인쇄 미리보기 + 인쇄 |
| `Ctrl+F2` | 인쇄 미리보기 (동일) |

## 🔨 가이드 실습 (25분)

### 실습 1: 보고서 인쇄 최적화 (12분)

**목표**: 학생 성적표를 완벽하게 인쇄 가능한 문서로 만듭니다.

1. **데이터 준비**:
   - 1행: 제목 행 (번호, 이름, 국어, 수학, 영어, 과학, 평균)
   - 2~31행: 학생 30명 데이터 입력 (가상 데이터)
   - 자동 채우기 활용: A2에 `1`, A3에 `2` → A31까지 드래그

2. **기본 서식 적용** (5강 복습):
   - 제목 행: 굵게 + 배경색 + 가운데 맞춤 + 테두리
   - 데이터 영역: 테두리 적용

3. **인쇄 설정**:
   - `Ctrl+P`로 현재 상태 확인 → 아마 2페이지로 나뉠 것
   - 페이지 레이아웃 → **방향: 가로** (열이 많으므로)
   - 페이지 레이아웃 → **여백: 좁게**
   - `Ctrl+P`로 다시 확인 → 한 페이지에 들어가나요?

4. **인쇄 제목 설정**:
   - 페이지 레이아웃 → 인쇄 제목 → 반복할 행: `$1:$1`
   - 다시 미리보기 → 2페이지에도 제목 행이 보이는지 확인

5. **바닥글 추가**:
   - 삽입 → 머리글/바닥글
   - 바닥글 가운데: "페이지 번호" 삽입
   - 바닥글 오른쪽: "날짜" 삽입


> ⚠️ **막힐 수 있는 포인트**:
> - **프린터가 없어서 인쇄를 못 해요** → 실제 인쇄할 필요 없습니다! `Ctrl+P`의 미리보기 화면에서 설정만 확인하면 됩니다
> - **30명의 데이터를 다 입력하기 힘들어요** → 이름은 "학생1", "학생2"로, 점수는 자동 채우기로 대략적으로 넣어도 됩니다. 인쇄 설정 연습이 목적입니다
> - **머리글/바닥글 편집 모드에서 나오는 법을 모르겠어요** → 시트의 아무 셀을 클릭하면 편집 모드에서 나옵니다

### 실습 2: 인쇄 영역과 페이지 나누기 (8분)

**목표**: 특정 부분만 인쇄하고, 페이지 나누기를 직접 조절합니다.

1. 이전 실습의 데이터에서 **A1:D16** 범위만 선택
2. 페이지 레이아웃 → **인쇄 영역 설정**
3. `Ctrl+P` → 선택한 범위만 인쇄됨을 확인
4. **인쇄 영역 지우기**로 해제

5. **페이지 나누기 연습**:
   - 보기 → **페이지 나누기 미리 보기**
   - 파란 점선(자동 나누기)을 드래그하여 원하는 위치로 이동
   - 보기 → 기본 보기로 돌아가기


> ⚠️ **막힐 수 있는 포인트**:
> - **인쇄 영역 설정 메뉴를 못 찾겠어요** → 페이지 레이아웃 탭 → "인쇄 영역" → "인쇄 영역 설정"입니다
> - **페이지 나누기 미리 보기에서 점선을 드래그할 수 없어요** → 파란 **실선**은 드래그 가능하지만, **점선**(자동 나누기)은 드래그하여 위치를 변경할 수 있습니다. 마우스가 ↔ 모양으로 바뀌는지 확인하세요

### 실습 3: 한 페이지 맞춤 인쇄 (5분)

**목표**: 넓은 표를 한 페이지에 축소해서 맞춰봅니다.

1. 열이 많은 표 (A~J열, 50행 정도) 만들기
2. `Ctrl+P` → 여러 페이지로 나뉨을 확인
3. 설정에서 **"시트를 한 페이지에 맞춤"** 선택
4. 미리보기 확인 → 글씨가 작아지지만 한 페이지에!
5. "모든 열을 한 페이지에 맞춤"으로 변경 → 너비만 맞추고 높이는 자동


> ⚠️ **막힐 수 있는 포인트**:
> - **"시트를 한 페이지에 맞춤" 옵션을 못 찾겠어요** → `Ctrl+P` → 설정 영역 하단에 "배율 없음" 드롭다운이 있습니다. 클릭하면 맞춤 옵션이 나타납니다
> - **한 페이지에 맞춤을 적용했는데 글씨가 너무 작아요** → 이는 정상입니다. 실무에서는 "모든 열을 한 페이지에 맞춤"으로 너비만 맞추고 행은 여러 페이지로 나누는 것이 좋습니다
