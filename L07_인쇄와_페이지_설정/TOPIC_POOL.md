# 🎯 화두 풀 — 인쇄와 페이지 설정

> 아래 주제를 골라 직접 실습해보세요!  
> 완벽한 인쇄 문서를 만드는 기술을 연습합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 보고서 인쇄 포맷 만들기 ⭐ 추천
> 이전 강의에서 만든 표(성적표, 견적서 등)를 완벽하게 인쇄 가능하도록 설정하세요.
- 적절한 방향(세로/가로) 선택
- 여백 "좁게" 적용
- 머리글에 문서 제목, 바닥글에 페이지 번호 삽입
- Ctrl+P로 미리보기 확인

### 2. 출석부 인쇄 양식
> 한 달 치 출석부를 인쇄용으로 만들어보세요.
- 1행: 날짜 (1~31일), A열: 학생 이름
- 가로 방향으로 설정
- 반복할 열: A열 (이름이 모든 페이지에)
- 바닥글: 인쇄 날짜 자동 삽입

---

## 🟡 도전 (조금 더 생각 필요)

### 3. 월간 보고서 템플릿
> 매월 제출하는 보고서 템플릿을 만들어보세요.
- 머리글: 왼쪽 "월간 보고서", 오른쪽 "2025년 X월"
- 바닥글: 가운데 "페이지 &P / &N" (현재 페이지/전체 페이지)
- 인쇄 제목으로 보고서 헤더를 모든 페이지에 반복
- PDF로 저장하여 결과 확인

### 4. 상품 카탈로그
> 10~20개 상품의 카탈로그를 인쇄용으로 만들어보세요.
- 상품별 이미지 영역(큰 셀), 상품명, 가격, 설명
- 한 페이지에 4~6개 상품이 들어가도록 행 높이 조절
- 페이지 나누기를 활용하여 상품이 중간에 잘리지 않도록

---

## 🔴 심화 (탐구해보기)

### 5. 인쇄 설정 비교 실험
> 같은 데이터를 다양한 인쇄 설정으로 비교해보세요.
- A4 세로 vs A4 가로
- 여백 보통 vs 좁게 vs 넓게
- 배율 100% vs 한 페이지 맞춤
- 각각 PDF로 저장하여 결과 비교

- 머리글에 "기밀", "대외비" 같은 워터마크를 넣으려면 어떻게 할까요?
- 흑백 인쇄할 때 조건부 서식의 색상이 구분이 안 되면 어떻게 해야 할까요?
