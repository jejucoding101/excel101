# 🎯 화두 풀 — 데이터 유효성 검사

> 아래 주제를 골라 직접 실습해보세요!

---

## 🟢 입문

### 1. 설문조사 입력 폼 ⭐ 추천
> 드롭다운과 입력 제한으로 진짜 설문조사 폼을 만드세요.

### 2. 주문서 입력 시스템
> 제품 드롭다운 → VLOOKUP 연동 → 수량 입력(1~999 제한) → 금액 자동 계산

---

## 🟡 도전

### 3. 학생 정보 등록 폼
> 학번, 이름, 학년(1~6), 반(1~10), 혈액형(A/B/O/AB) 등 다양한 유효성규칙을 적용하세요.

### 4. 종속 드롭다운 (연동)
> 탐구: 첫 드롭다운에서 "서울"을 선택하면, 두 번째 드롭다운에 서울의 구(강남구, 서초구...)만 표시되는 것이 가능할까요?
- 힌트: INDIRECT 함수 + 이름 정의

---

## 🔴 심화

### 5. 입력 폼 + 시트 보호
> 입력 셀만 편집 가능하고 나머지는 잠금 처리하세요.
- 검토 → 시트 보호
- 입력 셀은 셀 서식 → 보호 → "잠금" 해제

---
