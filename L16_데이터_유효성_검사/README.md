# 📌 16강: 데이터 유효성 검사 — 정확한 입력 보장하기

> **핵심 포인트**: 드롭다운 목록, 숫자 범위 제한, 입력 메시지로 정확한 데이터 입력을 유도합니다.

---

## 📖 이론 (20분)

### 데이터 유효성 검사란?

셀에 **입력할 수 있는 데이터를 제한**하여 잘못된 입력을 방지합니다.

```
유효성 검사 없이:              유효성 검사 적용 후:
사용자 마음대로 입력           드롭다운에서 선택!
├── "서울"                    ┌──────────┐
├── "서울시"    ← 오타!       │ 서울   ▼ │
├── "seoul"    ← 영어!       ├──────────┤
├── "seoull"   ← 오타!       │ 서울     │
└── ""         ← 빈칸!       │ 부산     │
                              │ 대구     │
                              │ 인천     │
                              └──────────┘
```

### 유효성 검사 설정 방법

1. 셀/범위 선택
2. **데이터 탭 → 데이터 유효성 검사**
3. 설정/입력 메시지/오류 알림 3개 탭 구성

### 드롭다운 목록 만들기 ⭐ (가장 많이 쓰는 기능!)

#### 방법 1: 직접 입력

1. 셀 선택 → 데이터 → 데이터 유효성 검사
2. 제한 대상: **목록** 선택
3. 원본에 직접 입력: `서울,부산,대구,인천,광주` (쉼표로 구분)

#### 방법 2: 셀 범위 참조 (추천!)

별도 시트에 목록을 정리해두고 참조합니다.

```
"목록" 시트:            "입력" 시트:
A열: 지역 목록          B열: 지역 ▼ (드롭다운)
서울                    → 원본: =목록!$A$1:$A$5
부산
대구
인천
광주
```

> 💡 **추천**: 방법 2가 목록 수정이 쉽습니다! 목록 시트에 항목을 추가하면 자동 반영.

### 숫자 범위 제한

| 설정 | 설명 | 실생활 예시 |
|------|------|-----------|
| 정수: 최소~최대 | 정수 범위 제한 | 수량: 1~100 |
| 소수: 범위 | 소수점 포함 | 할인율: 0~1.0 |
| 날짜: 범위 | 날짜 범위 | 입사일: 2020-01-01 이후 |
| 텍스트 길이 | 글자 수 제한 | 비밀번호: 8~20자 |

### 입력 메시지

셀을 선택하면 나타나는 안내 메시지:

```
┌──────────────────┐
│ 💬 입력 안내      │
│ → 1~100 사이의    │
│   정수를 입력하세요│
└──────────────────┘
```

설정: 유효성 검사 대화상자 → **입력 메시지** 탭 → 제목 + 메시지 입력

### 오류 알림

잘못된 데이터 입력 시 표시되는 경고:

| 스타일 | 동작 |
|--------|------|
| 중지 ⛔ | 입력 거부 (가장 엄격) |
| 경고 ⚠️ | 경고 후 사용자가 선택 가능 |
| 정보 ℹ️ | 안내 후 그냥 입력 가능 |

## 🔨 가이드 실습 (25분)

### 실습 1: 설문조사 입력 폼 (12분)

**목표**: 드롭다운과 입력 제한으로 정확한 데이터 수집 폼을 만듭니다.

1. **폼 설계**:
   ```
        A            B
   1행  항목         입력
   2행  이름         (자유 입력)
   3행  성별         (드롭다운: 남, 여)
   4행  나이         (정수: 10~100)
   5행  거주지역     (드롭다운: 서울, 부산, 대구, 인천, 광주, 대전, 울산, 세종)
   6행  만족도       (정수: 1~5)
   7행  연락처       (텍스트 길이: 11자리)
   ```

2. **성별 드롭다운** (B3):
   - 데이터 → 유효성 검사 → 목록 → `남,여` 입력

3. **나이 제한** (B4):
   - 유효성 검사 → 정수 → 최소값 10, 최대값 100
   - 입력 메시지: "10~100 사이의 나이를 입력하세요"
   - 오류 알림: 스타일 "중지", "10~100 사이만 가능합니다"

4. **거주지역 드롭다운** (B5):
   - 별도 영역(D열)에 지역 목록 입력 → 목록 원본을 셀 범위로 참조

5. **만족도 제한** (B6): 정수 1~5

6. **테스트**: 잘못된 값 입력 시도 → 오류 메시지 확인!

### 실습 2: VLOOKUP + 드롭다운 연동 (8분)

**목표**: 드롭다운에서 선택하면 관련 데이터가 자동으로 채워지는 시스템을 만듭니다.

1. 제품 목록 표 만들기 (15강에서 만든 것 활용)
2. 코드 입력 셀에 **제품 코드 드롭다운** 적용
3. 드롭다운에서 코드 선택 → VLOOKUP으로 제품명/단가 자동 표시!

### 실습 3: 입력 폼 완성 (5분)

- 모든 입력 셀에 적절한 유효성 검사 적용
- 배경색/테두리로 폼을 깔끔하게 꾸미기
- 테스트: 모든 항목을 올바르게/잘못 입력해보며 확인
