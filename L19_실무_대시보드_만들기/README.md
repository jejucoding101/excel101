# 📌 19강: 실무 대시보드 만들기

> **핵심 포인트**: 지금까지 배운 수식, 차트, 피벗, 조건부 서식, 유효성 검사를 통합하여 원페이지 대시보드를 만듭니다.

---

## 📖 이론 (20분)

### 대시보드란?

여러 데이터를 **한 화면에 요약**하여 핵심 정보를 한눈에 파악할 수 있는 시각적 보고서입니다.

```
┌──────────────────────────────────────────────────────┐
│  📊 매출 KPI 대시보드          기간: 2025년 1분기 ▼  │
├──────────┬──────────┬──────────┬─────────────────────┤
│ 💰총매출  │ 📦주문수 │ 👤고객수 │     월별 매출 추이   │
│ ₩4,500만 │  320건   │  180명   │    📈 <차트>        │
├──────────┴──────────┴──────────┤                     │
│                                │                     │
│  🥧 카테고리별 비율             │     ─────────────── │
│     <원형 차트>                 │                     │
│                                ├─────────────────────┤
├────────────────────────────────┤  🏆 상위 5개 제품    │
│  📋 지역별 매출 요약            │  1. 노트북  1,200만 │
│  ┌────┬──────┬──────┐         │  2. 모니터    800만 │
│  │지역│매출  │전월비│         │  3. 키보드    450만 │
│  ├────┼──────┼──────┤         │  4. 헤드셋    320만 │
│  │서울│2,100 │ ↑15% │         │  5. 마우스    280만 │
│  │부산│1,200 │ ↓ 3% │         │                     │
│  └────┴──────┴──────┘         │                     │
└────────────────────────────────┴─────────────────────┘
```

### 대시보드 구성 원칙

| 원칙 | 설명 |
|------|------|
| **한눈에** | 스크롤 없이 한 화면에 모든 핵심 정보 |
| **KPI 카드** | 핵심 지표를 큰 숫자로 강조 |
| **차트 2~3개** | 추이(꺾은선) + 비교(막대) + 비율(원형) |
| **필터** | 드롭다운이나 슬라이서로 기간/카테고리 전환 |
| **색상 규칙** | 3색 이내, 긍정=파랑/초록, 부정=빨강 |

### 대시보드 설계 3단계

```
1단계: 데이터 준비 (별도 시트에 원본 데이터)
2단계: 계산 시트 (SUMIF, VLOOKUP 등으로 요약 수치 계산)
3단계: 대시보드 시트 (차트 + KPI 카드 + 필터 배치)
```

### 이름 정의 (Named Range)

셀 범위에 **이름**을 붙여서 수식을 읽기 쉽게 만듭니다.

```
이름 정의 전:                    이름 정의 후:
=SUM(매출!B2:B100)               =SUM(전체매출)
=VLOOKUP(A1,제품!A:D,3,0)       =VLOOKUP(A1,제품마스터,3,0)
```

**방법**: 셀 범위 선택 → **이름 상자**(좌상단)에 이름 입력 → Enter
또는: 수식 탭 → 이름 정의

### 시트 보호

대시보드를 만들고 **실수로 수정되지 않도록** 보호합니다:
1. 입력 셀만 **잠금 해제** (Ctrl+1 → 보호 → "잠금" 체크 해제)
2. 검토 → **시트 보호** → 비밀번호 설정 (선택사항)

### 눈금선/머리글 숨기기

대시보드를 깔끔하게:
- 보기 → ☐ **눈금선** 체크 해제 → 배경이 깨끗한 흰색!
- 보기 → ☐ **머리글** 체크 해제 → A,B,C와 1,2,3 숨김

## 🔨 가이드 실습 (25분)

### 실습: 매출 KPI 대시보드 만들기 (25분)

**목표**: 3개 시트 구조로 실무 대시보드를 완성합니다.

##
> ⚠️ **막힐 수 있는 포인트**:
> - **다른 시트의 셀을 참조하는 법을 모르겠어요** → `=계산!B1`처럼 `시트이름!셀주소` 형식을 사용합니다. 또는 수식 입력 중에 직접 다른 시트의 셀을 클릭하면 자동으로 입력됩니다
> - **눈금선을 끄니 표가 안 보여요** → 눈금선은 인쇄에 영향을 주지 않는 보조선입니다. 필요한 곳에 테두리를 직접 그려 넣으세요
> - **차트를 다른 시트에서 만들고 대시보드로 옮기고 싶어요** → 차트를 선택 → 잘라내기(`Ctrl+X`) → 대시보드 시트 → 붙여넣기(`Ctrl+V`). 또는 차트 우클릭 → "차트 이동"
> - **KPI 카드에 배경색을 넣었는데 글자가 안 보여요** → 진한 배경에는 흰색 글꼴을 사용하세요. 셀 선택 → 홈 → 글꼴 색 → 흰색

## 1단계: 데이터 시트 ("원본데이터") — 5분

```
     A      B      C       D       E
1행  날짜   지역   카테고리 제품    매출
2행  1/5    서울   전자기기  노트북  1500000
3행  1/7    부산   사무용품  A4용지  25000
4행  1/10   서울   전자기기  마우스  35000
...  (20~30행의 가상 데이터)
```

#### 2단계: 계산 시트 ("계산") — 10분

```
KPI 계산:
A1: 총매출        B1: =SUM(원본데이터!E:E)
A2: 주문건수      B2: =COUNTA(원본데이터!A2:A100)
A3: 평균단가      B3: =AVERAGE(원본데이터!E:E)

지역별 요약:
A6: 지역    B6: 매출
A7: 서울    B7: =SUMIF(원본데이터!B:B,"서울",원본데이터!E:E)
A8: 부산    B8: =SUMIF(원본데이터!B:B,"부산",원본데이터!E:E)
...

카테고리별 요약:
(SUMIF로 카테고리별 매출 합계)
```

#### 3단계: 대시보드 시트 ("대시보드") — 10분

1. **눈금선 끄기**: 보기 → ☐ 눈금선

2. **제목 영역**: A1:H1 병합 → "📊 매출 KPI 대시보드" → 18pt, 굵게

3. **KPI 카드 3개**:
   - B3:C4 병합 → 총매출 값 참조 (`=계산!B1`) → 20pt, 통화 형식
   - D3:E4 병합 → 주문건수 → 20pt
   - F3:G4 병합 → 평균단가 → 20pt
   - 각 카드에 배경색, 테두리 적용

4. **차트 삽입**:
   - 계산 시트의 지역별 데이터로 **막대 차트** → 대시보드에 배치
   - 카테고리별 데이터로 **원형 차트** → 대시보드에 배치

5. **조건부 서식 활용**:
   - 목표 달성 여부에 따라 KPI 카드 색상 변경

6. **완성 및 테스트**:
   - 원본 데이터 수정 → 대시보드 숫자/차트가 자동 갱신되는지 확인!
