# 🎯 화두 풀 — 피벗 테이블

> 아래 주제를 골라 직접 실습해보세요!

---

## 🟢 입문

### 1. 인사팀 피벗 분석 보고서 ⭐ 추천
> 직원 20명의 데이터(부서/직급/성별/급여)를 피벗으로 다각도 분석하세요.
- 부서별 합계 → 부서×직급 교차 → 평균으로 변경 → 슬라이서 추가

### 2. 설문조사 결과 피벗 분석
> 설문 응답 데이터를 피벗으로 연령대별·성별 만족도를 교차 분석하세요.

---

## 🟡 도전

### 3. 매출 데이터 다차원 분석
> 100건 이상의 매출 데이터(날짜/지역/제품/금액)를 만들고
- 지역별 매출 합계, 제품별 매출 비율, 월별 추이를 피벗으로 분석하세요.
- 날짜 그룹화(월별)를 적용하세요.

### 4. 피벗 차트 대시보드
> 피벗 테이블 2개 + 피벗 차트 2개 + 슬라이서를 조합한 미니 대시보드를 만드세요.

---

## 🔴 심화

### 5. 계산 필드
> 피벗 테이블 → 분석 → "계산 필드"로 이익률(= 이익 / 매출)을 추가해보세요.
- 원본에는 없지만 피벗 내에서만 계산하는 필드!

## 💬 생각해볼 점 — 해답

**Q. 피벗 테이블은 SUMIF/COUNTIF를 대체할 수 있을까요?**
> **가능하지만 상황에 따라 다릅니다.**
> | 구분 | 피벗 테이블 | SUMIF/COUNTIF |
> |------|------------|---------------|
> | 갱신 | 수동 새로 고침 필요 | 자동 실시간 갱신 |
> | 유연성 | 드래그로 자유 변경 | 고정된 레이아웃 |
> | 사용 상황 | 탐색적 분석, 임시 보고 | 정해진 형식의 보고서 |

**Q. 원본 데이터를 수정하면 피벗에 자동 반영될까요?**
> **아니요.** 피벗 테이블을 우클릭 → **"새로 고침"**을 클릭해야 합니다. 또는 피벗 테이블 분석 탭 → 새로 고침. 파일을 열 때 자동 갱신되도록 설정하려면: 피벗 테이블 옵션 → 데이터 탭 → "파일을 열 때 데이터 새로 고침" 체크.

**Q. 피벗 vs 수식 — 각각 언제 사용하는 것이 좋을까요?**
> - **피벗**: "데이터를 어떤 관점으로 볼까?" 탐색할 때, 임시 분석, 큰 데이터 요약
> - **수식**: 매월 제출하는 고정 형식 보고서, 데이터 변경 시 자동 갱신이 필요할 때
