# 🎯 화두 풀 — 피벗 테이블

> 아래 주제를 골라 직접 실습해보세요!

---

## 🟢 입문

### 1. 인사팀 피벗 분석 보고서 ⭐ 추천
> 직원 20명의 데이터(부서/직급/성별/급여)를 피벗으로 다각도 분석하세요.
- 부서별 합계 → 부서×직급 교차 → 평균으로 변경 → 슬라이서 추가

### 2. 설문조사 결과 피벗 분석
> 설문 응답 데이터를 피벗으로 연령대별·성별 만족도를 교차 분석하세요.

---

## 🟡 도전

### 3. 매출 데이터 다차원 분석
> 100건 이상의 매출 데이터(날짜/지역/제품/금액)를 만들고
- 지역별 매출 합계, 제품별 매출 비율, 월별 추이를 피벗으로 분석하세요.
- 날짜 그룹화(월별)를 적용하세요.

### 4. 피벗 차트 대시보드
> 피벗 테이블 2개 + 피벗 차트 2개 + 슬라이서를 조합한 미니 대시보드를 만드세요.

---

## 🔴 심화

### 5. 계산 필드
> 피벗 테이블 → 분석 → "계산 필드"로 이익률(= 이익 / 매출)을 추가해보세요.
- 원본에는 없지만 피벗 내에서만 계산하는 필드!
