# 📌 18강: 피벗 테이블 — 데이터 요약의 끝판왕

> **핵심 포인트**: 피벗 테이블로 대량 데이터를 드래그&드롭만으로 다각도로 요약·분석하고, 슬라이서와 피벗 차트를 활용합니다.

---

## 📖 이론 (20분)

### 피벗 테이블이란?

대량의 **원본 데이터**를 다양한 관점에서 **요약하는 대화형 표**입니다.

```
원본 데이터 (수백~수만 행):              피벗 테이블 (자동 요약!):
┌────┬────┬────┬──────┐              ┌────────┬──────────┐
│날짜│지역│제품│매출   │     →        │ 지역   │ 매출 합계│
├────┼────┼────┼──────┤              ├────────┼──────────┤
│1/1 │서울│노트│150만 │     드래그    │ 서울   │ 4,500만 │
│1/1 │부산│마우│3.5만 │     &        │ 부산   │ 2,800만 │
│1/2 │서울│키보│8만   │     드롭!    │ 대구   │ 1,200만 │
│... │... │... │...   │              └────────┴──────────┘
│(200행)            │
└────┴────┴────┴──────┘              → 수식 하나 안 쓰고 완성!
```

> 🔑 **핵심**: 수식을 하나도 쓰지 않고, **마우스 드래그만으로** 데이터를 요약합니다!

### 피벗 테이블 만들기

1. 원본 데이터의 아무 셀 클릭
2. **삽입 → 피벗 테이블** (또는 `Alt+N+V`)
3. "테이블/범위" 확인 → 새 워크시트에 만들기 → 확인
4. 오른쪽에 **피벗 테이블 필드 목록** 패널이 나타남

### 4개 영역

```
┌─────────────────────────────────┐
│  피벗 테이블 필드               │
│                                 │
│  ☐ 날짜                        │
│  ☐ 지역     ← 필드 목록       │
│  ☐ 제품        (여기서 아래로   │
│  ☐ 매출        드래그!)        │
│                                 │
│  ┌──────────┬──────────┐       │
│  │ 🔽 필터  │ 📊 열    │       │
│  ├──────────┼──────────┤       │
│  │ 📋 행    │ Σ 값     │       │
│  └──────────┴──────────┘       │
└─────────────────────────────────┘
```

| 영역 | 역할 | 예시 |
|------|------|------|
| **행** (Row) | 세로 방향 그룹 | 지역별, 제품별 |
| **열** (Column) | 가로 방향 그룹 | 월별, 분기별 |
| **값** (Values) | 요약할 숫자 | 매출 합계, 주문 건수 |
| **필터** (Filter) | 전체 데이터 필터링 | 특정 연도만 |

### 값 필드 요약 방식

| 방식 | 설명 |
|------|------|
| **합계** (Sum) | 기본값, 숫자 합산 |
| **개수** (Count) | 데이터 개수 |
| **평균** (Average) | 평균값 |
| **최대** (Max) | 최대값 |
| **최소** (Min) | 최소값 |

변경법: 값 필드 클릭 → "값 필드 설정" → 요약 방식 변경

### 그룹화

날짜를 **년/분기/월** 단위로 자동 묶을 수 있습니다:
- 날짜 셀 우클릭 → 그룹화 → 월/분기/년 선택

### 슬라이서 (Slicer) ⭐

**시각적인 필터 버튼**입니다. 클릭만으로 데이터를 필터링!

```
┌──────────┐  ┌──────────┐
│ [서울]   │  │ [노트북] │
│ [부산]   │  │ [마우스] │
│ [대구]   │  │ [키보드] │
│ [인천]   │  │ [모니터] │
└──────────┘  └──────────┘
   지역 슬라이서   제품 슬라이서
```

삽입 → 슬라이서 → 원하는 필드 선택
- Ctrl 클릭으로 여러 항목 동시 선택 가능!

### 피벗 차트

피벗 테이블에서 바로 차트를 만들 수 있습니다:
- 피벗 테이블 선택 → 삽입 → 피벗 차트

> 슬라이서를 클릭하면 **피벗 테이블과 피벗 차트가 동시에** 갱신됩니다!

### ⌨️ 이번 강의 필수 단축키

| 단축키 | 기능 |
|--------|------|
| `Alt+N+V` | 피벗 테이블 삽입 |
| 필드를 영역에 드래그 | 피벗 구성 변경 |
| 우클릭 → 새로 고침 | 원본 변경 반영 |

## 🔨 가이드 실습 (25분)

### 실습 1: 인사팀 피벗 분석 보고서 (15분)

**목표**: 직원 데이터를 피벗 테이블로 다각도 분석합니다.

1. **원본 데이터 준비** (20행 이상):
   ```
        A      B      C       D       E
   1행  이름   부서   직급    성별    급여
   2행  홍길동 영업   대리    남      350
   3행  김영희 개발   과장    여      450
   4행  이민수 영업   사원    남      280
   5행  박지은 인사   대리    여      340
   6행  정다빈 개발   대리    남      380
   7행  강하늘 영업   과장    여      420
   8행  최유진 개발   사원    남      300
   9행  한소희 인사   과장    여      440
   ...  (20행까지 다양하게)
   ```

2. **피벗 테이블 생성**:
   - 데이터 아무 셀 클릭 → 삽입 → 피벗 테이블 → 새 워크시트 → 확인

3. **분석 1: 부서별 급여 합계**
   - "부서" → **행** 영역으로 드래그
   - "급여" → **값** 영역으로 드래그
   - 결과: 부서별 급여 합계가 자동으로!

4. **분석 2: 부서별·직급별 교차 분석**
   - "직급" → **열** 영역으로 드래그
   - 결과: 부서(행) × 직급(열)의 급여 합계 매트릭스!

5. **분석 3: 평균 급여로 변경**
   - "합계:급여" 클릭 → 값 필드 설정 → **평균**으로 변경

6. **슬라이서 추가**:
   - 삽입 → 슬라이서 → "성별" 선택
   - "남" 클릭 → 남성 직원만 필터! / "여" 클릭 → 여성만!

### 실습 2: 피벗 차트 만들기 (5분)

1. 피벗 테이블 선택 → 삽입 → 피벗 차트 → 세로 막대형
2. 슬라이서 클릭 → 차트도 함께 변하는지 확인!

### 실습 3: 필드 바꿔보기 (5분)

1. 행 영역에 "부서" 대신 "성별"을 넣어보세요
2. 열 영역의 필드를 바꿔보세요
3. 값 영역에 "급여" 외에 "이름"을 넣으면? → **개수**로 자동 집계!
4. 다양한 조합을 시도해보세요 — **이것이 피벗의 매력!**
