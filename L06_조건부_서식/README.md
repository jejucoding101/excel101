# 📌 6강: 조건부 서식으로 데이터 강조하기

> **핵심 포인트**: 조건부 서식을 사용하여 특정 조건을 만족하는 셀의 서식을 자동으로 변경합니다.

---

## 📖 이론 (20분)

### 조건부 서식이란?

**조건을 만족하는 셀만** 자동으로 서식(색상, 아이콘 등)을 적용하는 기능입니다.

```
일반 서식:                    조건부 서식 적용 후:
┌──────┬──────┐              ┌──────┬──────┐
│ 이름 │ 점수 │              │ 이름 │ 점수 │
├──────┼──────┤              ├──────┼──────┤
│ 홍길동│  95  │              │ 홍길동│ 🟦95 │ ← 90점 이상: 파랑
│ 김철수│  72  │              │ 김철수│  72  │
│ 이영희│  48  │              │ 이영희│ 🟥48 │ ← 60점 미만: 빨강
│ 박민수│  88  │              │ 박민수│  88  │
│ 정수정│  35  │              │ 정수정│ 🟥35 │ ← 60점 미만: 빨강
└──────┴──────┘              └──────┴──────┘
```

> 💡 **핵심**: 데이터가 바뀌면 서식도 **자동으로 변경**됩니다! 한 번 설정하면 끝!

### 조건부 서식 메뉴 찾기

**홈 탭 → 조건부 서식** (또는 단축키 `Alt+H+L`)

5가지 주요 유형이 있습니다:

### 1. 셀 강조 규칙

특정 **조건에 맞는 셀**에 서식을 적용합니다.

| 규칙 | 설명 | 예시 |
|------|------|------|
| 보다 큼 | 특정 값보다 큰 셀 강조 | 점수 > 90 |
| 보다 작음 | 특정 값보다 작은 셀 강조 | 재고 < 10 |
| 사이에 있음 | 범위 안의 셀 강조 | 70~89 |
| 같음 | 특정 값과 같은 셀 | "불합격" |
| 텍스트 포함 | 특정 텍스트가 있는 셀 | "VIP" 포함 |
| 날짜 | 날짜 조건 | 지난 7일간 |
| 중복 값 | 중복된 값 찾기 | 중복 이름 찾기 |

**적용 방법**:
1. 범위 선택 (예: B2:B20)
2. 홈 → 조건부 서식 → 셀 강조 규칙 → "보다 큼..."
3. 값 입력 (예: 90) → 서식 선택 (예: 밝은 빨강 채우기) → 확인

### 2. 상위/하위 규칙

데이터에서 **상위/하위 N개** 또는 **N%**를 강조합니다.

| 규칙 | 설명 |
|------|------|
| 상위 10개 항목 | 가장 큰 10개 |
| 상위 10% | 상위 10%에 해당하는 값 |
| 하위 10개 항목 | 가장 작은 10개 |
| 평균 초과 | 평균보다 큰 값 |
| 평균 미만 | 평균보다 작은 값 |

### 3. 데이터 막대 📊

셀 안에 **막대 그래프**를 표시합니다. 숫자 크기를 시각적으로 비교할 때 유용합니다.

```
┌──────┬──────────────────┐
│ 매출 │                  │
├──────┼──────────────────┤
│ 50   │ ████░░░░░░░░░░░░ │
│ 100  │ ████████░░░░░░░░ │
│ 150  │ ████████████░░░░ │
│ 200  │ ████████████████ │
└──────┴──────────────────┘
```

### 4. 색조 (Color Scale) 🌈

셀의 값에 따라 **배경색이 점진적으로** 변합니다 (히트맵처럼).

```
낮은 값 ←──── 중간 ────→ 높은 값
🟥 빨강   🟨 노랑    🟩 초록
```

### 5. 아이콘 집합 🔴🟡🟢

셀에 **아이콘**을 추가하여 상태를 시각적으로 표시합니다.

```
┌──────┬──────┬──────┐
│ 이름 │ 달성률│ 상태 │
├──────┼──────┼──────┤
│ 홍길동│ 95%  │ 🟢   │  ← 67% 이상: 녹색
│ 김철수│ 50%  │ 🟡   │  ← 33~67%: 노랑
│ 이영희│ 20%  │ 🔴   │  ← 33% 미만: 빨강
└──────┴──────┴──────┘
```

### 조건부 서식 관리

여러 규칙이 겹칠 때 관리 방법:
- **홈 → 조건부 서식 → 규칙 관리** → 규칙 우선순위 확인/변경/삭제
- 규칙 삭제: 범위 선택 → 홈 → 조건부 서식 → **규칙 지우기**

### ⌨️ 이번 강의 필수 단축키

| 단축키 | 기능 |
|--------|------|
| `Alt+H+L` | 조건부 서식 메뉴 |
| `Ctrl+1` | 셀 서식 대화상자 |

## 🔨 가이드 실습 (25분)

### 실습 1: 성적표 자동 색칠하기 (12분)

**목표**: 점수에 따라 셀 배경색이 자동으로 바뀌는 성적표를 만듭니다.

1. **데이터 입력**:
   ```
        A        B      C      D      E
   1행  이름     국어   수학   영어   과학
   2행  홍길동    95     78     88     92
   3행  김영희    67     92     74     85
   4행  이민수    45     56     63     71
   5행  박지은    83     91     97     88
   6행  정다빈    72     48     55     60
   7행  강하늘    90     85     82     93
   8행  최유진    38     42     51     45
   ```

2. **90점 이상 파란색 강조**:
   - B2:E8 범위 선택
   - 홈 → 조건부 서식 → 셀 강조 규칙 → 보다 큼 → `89` → 진한 파란 텍스트, 밝은 파랑 배경 → 확인

3. **60점 미만 빨간색 강조**:
   - 같은 범위 선택
   - 홈 → 조건부 서식 → 셀 강조 규칙 → 보다 작음 → `60` → 진한 빨간 텍스트, 밝은 빨강 배경 → 확인

4. **결과 확인**: 90점 이상은 파랑, 60점 미만은 빨강으로 자동 표시!
   - 값을 바꿔보세요 → 색상이 자동으로 업데이트 되나요? ✅

### 실습 2: 데이터 막대와 색조 (8분)

**목표**: 매출 데이터를 데이터 막대와 색조로 시각화합니다.

1. **매출 데이터 입력**:
   ```
        A          B
   1행  지점       매출(만원)
   2행  강남점     12500
   3행  홍대점     9800
   4행  명동점     15200
   5행  신촌점     7300
   6행  이태원점   11000
   7행  잠실점     8500
   ```

2. **데이터 막대 적용**:
   - B2:B7 선택 → 홈 → 조건부 서식 → 데이터 막대 → 파란 데이터 막대 선택
   - 숫자 크기 비교가 한눈에!

3. **다른 열에 같은 데이터 복사 → 색조 적용**:
   - B열 데이터를 C열에 복사
   - C2:C7 선택 → 홈 → 조건부 서식 → 색조 → 빨강-노랑-초록 선택
   - 두 가지 시각화를 비교해보세요!

### 실습 3: 아이콘 집합 활용하기 (7분)

**목표**: 목표 달성률에 신호등 아이콘을 적용합니다.

1. **데이터 입력**:
   ```
        A          B
   1행  프로젝트   달성률
   2행  프로젝트A  0.95
   3행  프로젝트B  0.72
   4행  프로젝트C  0.45
   5행  프로젝트D  0.88
   6행  프로젝트E  0.30
   ```

2. **백분율 표시 형식**: B2:B6 → `Ctrl+Shift+5`

3. **아이콘 집합 적용**:
   - B2:B6 선택 → 홈 → 조건부 서식 → 아이콘 집합 → 3가지 신호등 선택
   - 결과: 높은 달성률 🟢, 중간 🟡, 낮은 달성률 🔴
