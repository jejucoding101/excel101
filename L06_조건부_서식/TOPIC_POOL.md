# 🎯 화두 풀 — 조건부 서식

> 아래 주제를 골라 직접 실습해보세요!  
> 조건부 서식으로 데이터를 자동으로 시각화합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 성적표 자동 강조 ⭐ 추천
> 학생 10명의 과목별 점수를 입력하고 조건부 서식을 적용하세요.
- 90점 이상: 파란 배경 (우수)
- 60점 미만: 빨간 배경 (위험)
- 점수를 바꿔보면서 색이 자동으로 변하는지 확인!

### 2. 출석 현황판
> 학생 이름과 출결 상태(출석/결석/지각)를 입력한 표에 조건부 서식을 적용하세요.
- "출석" = 초록 배경
- "결석" = 빨강 배경
- "지각" = 노랑 배경
- 힌트: 셀 강조 규칙 → "같음" 사용

### 3. 기온 히트맵
> 1주일간 시간대별(9시~18시) 기온 데이터를 입력하고 색조를 적용하세요.
- 7열(요일) × 10행(시간대) 표 만들기
- 가상의 기온(예: 5~25도)을 입력
- 색조: 파랑(차가움) → 빨강(더움) 적용

---

## 🟡 도전 (조금 더 생각 필요)

### 4. 가계부 지출 경고 시스템
> 월별 가계부에서 일정 금액 이상 지출에 자동 경고를 만드세요.
- 지출 50,000원 이상: 노란 배경
- 지출 100,000원 이상: 빨간 배경
- 데이터 막대로 지출 규모 시각화

### 5. 프로젝트 진행률 대시보드
> 5개 프로젝트의 진행률(0~100%)에 조건부 서식을 적용하세요.
- 아이콘 집합: 🟢🟡🔴 신호등
- 데이터 막대: 진행률 막대
- 두 가지를 나란히 비교해보세요

---

## 🔴 심화 (탐구해보기)

### 6. 수식을 사용한 조건부 서식
> 조건부 서식 → 새 규칙 → "수식을 사용하여 서식을 지정할 셀 결정"을 탐구해보세요.
- 주말(토/일) 행 전체에 회색 배경 적용
- 수식 예: `=WEEKDAY(A2,2)>5` (5보다 크면 주말)
- 이 기능은 아주 강력합니다! 나중에 IF 함수를 배우면 더 활용할 수 있어요.

## 💬 생각해볼 점 — 해답

**Q. "보다 큼 89"와 "보다 큼 90"은 결과가 다를까요?**
> **다릅니다!**
> - "보다 큼 89" → 89 초과, 즉 **90점 이상** 포함
> - "보다 큼 90" → 90 초과, 즉 **91점 이상**만 포함 (90점 제외!)
> 정확히 90 이상을 강조하려면 "크거나 같음(>=)" 조건을 사용하거나, "보다 큼 89"를 사용하세요.

**Q. 조건부 서식을 너무 많이 적용하면 어떤 문제가 생길 수 있을까요?**
> ① 파일 **용량 증가** 및 **작동 속도 저하**
> ② 규칙 간의 우선순위 충돌로 **예상치 못한 결과** 발생
> ③ 규칙이 많아지면 **유지보수가 어려움**
> "조건부 서식 규칙 관리자"로 주기적으로 정리하는 것이 좋습니다.

**Q. 데이터 막대와 색조 중 어떤 상황에서 어떤 것이 더 효과적일까요?**
> - **데이터 막대**: 값의 절대적 크기를 비교할 때 (예: 지점별 매출 비교)
> - **색조**: 전체 분포에서 상대적 위치를 볼 때 (예: 기온 히트맵, 밀도 분포)

**Q. 중복 값 찾기 기능은 언제 유용할까요?**
> 예: 같은 고객이 두 번 등록된 경우, 동일 상품이 중복 입력된 경우. 조건부 서식 → 셀 강조 규칙 → **중복 값**을 선택하면 한눈에 찾을 수 있습니다.
