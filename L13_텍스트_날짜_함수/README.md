# 📌 13강: 텍스트와 날짜 함수

> **핵심 포인트**: LEFT, RIGHT, MID로 텍스트를 쪼개고 CONCATENATE/TEXTJOIN으로 합치며, TODAY, DATEDIF로 날짜를 다룹니다.

---

## 📖 이론 (20분)

### 텍스트 함수 — 문자열 가공

#### 문자열 추출 함수

| 함수 | 기능 | 예시 (A1 = "홍길동") |
|------|------|---------------------|
| `LEFT(텍스트, 글자수)` | 왼쪽에서 N글자 | `=LEFT(A1, 1)` → "홍" |
| `RIGHT(텍스트, 글자수)` | 오른쪽에서 N글자 | `=RIGHT(A1, 2)` → "길동" |
| `MID(텍스트, 시작, 글자수)` | 중간에서 N글자 | `=MID(A1, 2, 1)` → "길" |
| `LEN(텍스트)` | 글자 수 | `=LEN(A1)` → 3 |

#### 실전 예시: 주민번호에서 정보 추출

```
A1 = "901215-1234567"

성별코드: =MID(A1, 8, 1)    → "1" (남성)
생년:    =LEFT(A1, 2)       → "90"
생월:    =MID(A1, 3, 2)     → "12"
생일:    =MID(A1, 5, 2)     → "15"
```

#### 문자열 결합 함수

| 함수 | 예시 | 결과 |
|------|------|------|
| `CONCATENATE(A, B)` | `=CONCATENATE("홍","길동")` | "홍길동" |
| `&` 연산자 | `=A1&" "&B1` | "홍 길동" |
| `TEXTJOIN(구분자, 빈칸무시, 범위)` | `=TEXTJOIN(", ", TRUE, A1:A3)` | "홍, 길, 동" |

> 💡 `&` 연산자가 가장 간편합니다! 문자 사이에 공백을 넣으려면 `" "`을 추가하세요.

#### 문자열 정리 함수

| 함수 | 기능 | 예시 |
|------|------|------|
| `TRIM(텍스트)` | 앞뒤 공백 제거 | `=TRIM("  홍길동  ")` → "홍길동" |
| `UPPER(텍스트)` | 대문자로 | `=UPPER("abc")` → "ABC" |
| `LOWER(텍스트)` | 소문자로 | `=LOWER("ABC")` → "abc" |
| `PROPER(텍스트)` | 각 단어 첫 글자 대문자 | `=PROPER("john doe")` → "John Doe" |
| `SUBSTITUTE(텍스트, 이전, 새로운)` | 문자 치환 | `=SUBSTITUTE(A1, "-", "")` → 하이픈 제거 |

### TEXT 함수 — 숫자/날짜를 원하는 형식의 문자로

```
=TEXT(값, 형식코드)

=TEXT(45678, "yyyy년 mm월 dd일")  → "2025년 01월 15일"
=TEXT(3500000, "#,##0원")        → "3,500,000원"
=TEXT(0.856, "0.0%")             → "85.6%"
```

### 날짜 함수

#### 현재 날짜/시간

| 함수 | 기능 | 특징 |
|------|------|------|
| `TODAY()` | 오늘 날짜 | 파일 열 때마다 갱신 |
| `NOW()` | 현재 날짜+시간 | 파일 열 때마다 갱신 |

> ⚠️ TODAY()와 NOW()는 **고정 값이 아닙니다!** 내일 열면 내일 날짜로 바뀝니다.
> 고정 날짜를 원하면 `Ctrl+;` (오늘 날짜)를 사용하세요.

#### 날짜 정보 추출

| 함수 | 기능 | 예시 (2025-03-15) |
|------|------|-------------------|
| `YEAR(날짜)` | 연도 | 2025 |
| `MONTH(날짜)` | 월 | 3 |
| `DAY(날짜)` | 일 | 15 |
| `WEEKDAY(날짜)` | 요일 번호 | (1=일~7=토) |

#### 날짜 계산

```
날짜끼리 빼기:
=A2-A1                    → 두 날짜 사이의 일수 차이

DATEDIF: 기간 계산 (숨겨진 함수!)
=DATEDIF(시작일, 종료일, 단위)

단위:
"Y" = 년수    "M" = 개월수    "D" = 일수

예시:
=DATEDIF("2000-05-15", TODAY(), "Y")  → 만 나이 (년)
=DATEDIF("2023-01-01", TODAY(), "M")  → 경과 개월수
```

#### EDATE와 EOMONTH

```
=EDATE(시작일, 개월수)    → N개월 후의 날짜
=EOMONTH(시작일, 개월수)  → N개월 후의 월말 날짜

예시:
=EDATE("2025-01-15", 3)   → 2025-04-15
=EOMONTH("2025-01-15", 0)  → 2025-01-31 (이번 달 말일)
```

## 🔨 가이드 실습 (25분)

### 실습 1: 전화번호부 정리기 (10분)

**목표**: 텍스트 함수로 전화번호를 분리하고 합치는 연습을 합니다.

1. **데이터 입력**:
   ```
        A                B          C       D         E
   1행  전화번호(통째로)  앞자리    중간    뒷자리    정리된 번호
   2행  01012345678      (수식)    (수식)  (수식)    (수식)
   3행  01098765432      (수식)    (수식)  (수식)    (수식)
   4행  01055556666      (수식)    (수식)  (수식)    (수식)
   ```

2. **전화번호 분리 수식**:
   - B2: `=LEFT(A2, 3)` → 010
   - C2: `=MID(A2, 4, 4)` → 1234
   - D2: `=RIGHT(A2, 4)` → 5678

3. **정리된 번호 합치기**:
   - E2: `=B2&"-"&C2&"-"&D2` → 010-1234-5678

4. 채우기 핸들로 아래 행에 복사


> ⚠️ **막힐 수 있는 포인트**:
> - **LEFT, MID, RIGHT 함수의 숫자 인수가 헷갈려요** → LEFT(텍스트, 왼쪽부터 몇 글자), MID(텍스트, 시작위치, 몇 글자), RIGHT(텍스트, 오른쪽부터 몇 글자)입니다
> - **전화번호가 숫자로 인식되어 앞의 0이 사라졌어요** → 원본 데이터(A열)를 입력할 때 앞에 작은따옴표(`'`)를 붙이거나, 셀 서식을 "텍스트"로 먼저 설정하세요
> - **`&` 연산자로 텍스트를 합칠 때 하이픈이 안 들어가요** → `&"-"&`처럼 하이픈도 큰따옴표로 감싸서 문자열로 지정해야 합니다

### 실습 2: 만 나이 및 D-day 계산기 (10분)

**목표**: 날짜 함수로 나이와 기념일까지 남은 일수를 계산합니다.

```
     A           B            C            D
1행  이름        생년월일     만 나이       다음 생일까지
2행  홍길동      1995-05-20   (수식)       (수식)
3행  김영희      2000-12-03   (수식)       (수식)
4행  이민수      1998-08-15   (수식)       (수식)
```

- C2 (만 나이): `=DATEDIF(B2, TODAY(), "Y")`
- D2 (다음 생일까지): 
  ```
  =IF(DATE(YEAR(TODAY()),MONTH(B2),DAY(B2))>=TODAY(),
     DATE(YEAR(TODAY()),MONTH(B2),DAY(B2))-TODAY(),
     DATE(YEAR(TODAY())+1,MONTH(B2),DAY(B2))-TODAY())
  ```
  (조금 복잡하지만, IF+DATE+TODAY 조합!)

> 💡 간단 버전: D-day 날짜를 직접 입력한 셀에서 TODAY()를 빼는 것도 좋습니다.
> 예: `="2025-12-25"-TODAY()` → 크리스마스까지 남은 일수


> ⚠️ **막힐 수 있는 포인트**:
> - **DATEDIF 함수가 자동 완성에 안 나타나요** → 정상입니다! DATEDIF는 "숨겨진 함수"로 자동 완성 목록에는 없지만 직접 입력하면 정상 작동합니다
> - **다음 생일 수식이 너무 복잡해요** → 간단 버전으로 시작하세요: 특정 날짜까지 남은 일수는 `="2025-12-25"-TODAY()` 한 줄로 구할 수 있습니다
> - **TODAY()가 매일 바뀌나요?** → 네! TODAY()는 파일을 열 때마다 오늘 날짜로 자동 갱신됩니다

### 실습 3: TRIM과 SUBSTITUTE 활용 (5분)

**목표**: 지저분한 데이터를 깔끔하게 정리합니다.

```
     A(원본)                    B(정리)
1행  "  홍 길 동  "            =TRIM(A1)              → "홍 길 동"
2행  "010-1234-5678"           =SUBSTITUTE(A2,"-","") → "01012345678"
3행  "seoul KOREA"             =PROPER(A3)            → "Seoul Korea"
```


> ⚠️ **막힐 수 있는 포인트**:
> - **TRIM을 적용했는데 가운데 공백이 안 사라져요** → TRIM은 앞뒤 공백만 제거하고, 단어 사이의 공백 1개는 유지합니다. "홍 길 동"의 공백을 모두 제거하려면 `=SUBSTITUTE(A1," ","")`을 사용하세요
> - **PROPER 함수가 한글에 적용이 안 돼요** → PROPER은 영문 전용 함수입니다. 한글에는 효과가 없습니다
