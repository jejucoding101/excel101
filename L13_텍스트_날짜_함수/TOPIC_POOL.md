# 🎯 화두 풀 — 텍스트와 날짜 함수

> 아래 주제를 골라 직접 실습해보세요!  
> 텍스트 가공과 날짜 계산을 활용합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 전화번호부 정리기 ⭐ 추천
> 하이픈 없이 입력된 전화번호를 LEFT/MID/RIGHT와 &로 "010-XXXX-XXXX" 형식으로 변환하세요.

### 2. D-day 계산기
> 중요한 기념일(시험, 생일, 여행 등)까지 남은 일수를 TODAY()를 활용하여 계산하세요.
- 5가지 이상의 D-day를 등록

---

## 🟡 도전 (조금 더 생각 필요)

### 3. 이메일 주소 분석기
> 이메일 주소에서 아이디(@앞)와 도메인(@뒤)을 분리하세요.
- 힌트: `=LEFT(A1, FIND("@",A1)-1)` → 아이디
- 힌트: `=MID(A1, FIND("@",A1)+1, 100)` → 도메인
- FIND 함수: 특정 문자의 위치를 찾는 함수!

### 4. 근무일수 / 근속연수 계산기
> 직원의 입사일에서 근무일수, 근속연수를 자동 계산하세요.
- DATEDIF로 근속 년/월/일 분리
- TEXT로 "X년 Y개월 Z일" 형식 표시

---

## 🔴 심화 (탐구해보기)

### 5. 주민등록번호 마스킹
> 주민번호 뒷자리를 "******"으로 마스킹하세요.
- `=LEFT(A1,8)&"******"` → "901215-1******"
- REPT 함수: `=LEFT(A1,8)&REPT("*",6)` → 같은 결과!

## 💬 생각해볼 점 — 해답

**Q. 한글 1글자와 영어 1글자는 LEN에서 모두 1로 계산될까요?**
> **네, 둘 다 1**로 계산됩니다. LEN은 **문자(character)** 수를 세며, 한글이든 영어든 1글자 = 1로 납니다. 단, 바이트(byte) 기준으로는 영어 1자 = 1바이트, 한글 1자 = 2~3바이트로 다릅니다. (LENB 함수를 사용하면 바이트 수를 셀 수 있습니다)

**Q. DATEDIF는 왜 함수 마법사에 안 나타날까요?**
> Microsoft가 공식적으로 지원하지 않는 **레거시(legacy) 함수**이기 때문입니다. Lotus 1-2-3과의 호환성을 위해 남겨두었지만, 공식 문서화하지 않기로 결정했습니다. 직접 입력하면 작동하지만, 향후 버전에서 제거될 수도 있습니다.

**Q. TEXT 함수와 셀 서식(5강)의 표시 형식은 어떤 차이가 있을까요?**
> - **셀 서식**: 같은 셀의 표시 방식만 바꿈. 수식에서는 원래 값으로 계산됨
> - **TEXT 함수**: 값을 **문자열로 변환**하여 **다른 셀**에 출력. 계산에 사용 불가
> 예: `=TEXT(A1,"yyyy년 mm월")` → "2025년 03월" 이라는 텍스트 생성. 이 결과는 날짜가 아닌 문자이므로 날짜 계산에 사용 불가합니다.
