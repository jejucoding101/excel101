# 🎯 화두 풀 — 텍스트와 날짜 함수

> 아래 주제를 골라 직접 실습해보세요!  
> 텍스트 가공과 날짜 계산을 활용합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 전화번호부 정리기 ⭐ 추천
> 하이픈 없이 입력된 전화번호를 LEFT/MID/RIGHT와 &로 "010-XXXX-XXXX" 형식으로 변환하세요.

### 2. D-day 계산기
> 중요한 기념일(시험, 생일, 여행 등)까지 남은 일수를 TODAY()를 활용하여 계산하세요.
- 5가지 이상의 D-day를 등록

---

## 🟡 도전 (조금 더 생각 필요)

### 3. 이메일 주소 분석기
> 이메일 주소에서 아이디(@앞)와 도메인(@뒤)을 분리하세요.
- 힌트: `=LEFT(A1, FIND("@",A1)-1)` → 아이디
- 힌트: `=MID(A1, FIND("@",A1)+1, 100)` → 도메인
- FIND 함수: 특정 문자의 위치를 찾는 함수!

### 4. 근무일수 / 근속연수 계산기
> 직원의 입사일에서 근무일수, 근속연수를 자동 계산하세요.
- DATEDIF로 근속 년/월/일 분리
- TEXT로 "X년 Y개월 Z일" 형식 표시

---

## 🔴 심화 (탐구해보기)

### 5. 주민등록번호 마스킹
> 주민번호 뒷자리를 "******"으로 마스킹하세요.
- `=LEFT(A1,8)&"******"` → "901215-1******"
- REPT 함수: `=LEFT(A1,8)&REPT("*",6)` → 같은 결과!
