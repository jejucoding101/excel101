# 🎯 화두 풀 — 데이터 정렬과 필터

> 아래 주제를 골라 직접 실습해보세요!

---

## 🟢 입문 (바로 시도 가능)

### 1. 쇼핑몰 주문 데이터 분석 ⭐ 추천
> 20건 이상의 주문 데이터를 만들고 다양한 정렬과 필터를 적용하세요.

### 2. 도서 대출 관리
> 도서관 대출 기록(도서명, 저자, 장르, 대출일, 반납일)을 정렬/필터하세요.
- 장르별 필터, 대출일 기준 정렬

---

## 🟡 도전

### 3. 학생 성적 다중 정렬
> 30명 학생의 성적 데이터를 만들고 반별→총점 내림차순 정렬하세요.

### 4. 고객 데이터 분석
> 고객 명단(이름, 지역, 가입일, 등급, 구매금액)에서 필터로 VIP 고객만 추출하세요.

---

## 🔴 심화

### 5. 고급 필터 도전
> 데이터 → 고급 필터를 사용하여 조건 범위를 별도로 지정해보세요.
- 조건 범위란? 제목행+조건을 별도 영역에 만들어 복잡한 조건을 적용하는 기능

---
