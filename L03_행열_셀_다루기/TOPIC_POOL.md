# 🎯 화두 풀 — 행, 열, 셀 다루기

> 아래 주제를 골라 직접 실습해보세요!
> 행/열 삽입·삭제, 셀 병합, 틀 고정을 여행사 실무에 활용합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 해외여행 일정표 ⭐ 추천
> 4박 5일 해외여행 일정표를 엑셀로 만들어보세요.
- 1행: 날짜(1일차~5일차), A열: 시간대(오전/오후/저녁) → 1행과 A열을 **틀 고정**하면 편리!
- 국가명 행은 **셀 병합**으로 넓게 표시 (예: A3~G3 병합 → "🇹🇭 태국 방콕")
- 주요 일정(관광지, 식사, 숙박)을 표로 채우고, 열 너비를 내용에 맞게 조절

### 2. 버스 좌석 배치도
> 여행 투어 버스의 좌석 배치도를 셀 병합으로 만들어보세요.
- 운전석, 가이드석, 화장실 구역을 셀 병합으로 표현
- 나머지 셀에 좌석 번호 입력 (1A, 1B, 2A, 2B...)
- 행/열 크기를 조절하여 실제 배치도처럼 보이게 만들기

### 3. 여행 상품 비교표
> 3개 여행 상품을 한눈에 비교하는 표를 만들어보세요.
- 상품명 행을 병합, 그 아래 항목(일정, 가격, 포함사항, 불포함사항)을 나열
- A열(항목명)을 틀 고정하여 오른쪽으로 스크롤해도 항목이 보이게!

---

## 🟡 도전 (조금 더 생각 필요)

### 4. 주간 가이드 배정표
> 이번 주 가이드별 투어 일정을 관리하는 표를 만들어보세요.
- 열: 월~일(7열), 행: 가이드 이름
- 투어가 없는 칸에 행을 추가하여 공지사항 입력
- 중간에 취소된 투어 행을 삭제하는 연습 (Ctrl+Z로 되돌리기도 확인!)
- 틀 고정으로 가이드 이름과 요일이 항상 보이게 설정

### 5. 호텔 객실 배정표
> 단체 여행 투어의 객실 배정표를 만들어보세요.
- 객실 유형(싱글/더블/트윈)별로 셀 병합하여 구역 표시
- 각 객실에 배정된 고객명 입력
- 급히 변경된 배정은 행 삽입으로 처리

## 🔴 심화 (탐구해보기)

### 6. 행선지별 다중 시트 관리
> 주요 여행지(방콕, 발리, 오사카, 파리)별로 시트를 나누어 투어 데이터를 관리해보세요.
- 각 시트: 일정표 + 포함 호텔 목록
- 1행(날짜) + A열(시간대) 틀 고정 각 시트별 적용
- 시트 탭 색상으로 대륙별 구분 (동남아=초록, 일본=빨강, 유럽=파랑)
- 시트 이름을 행선지명으로 변경 (탭 더블클릭)

## 💬 생각해볼 점 — 해답

**Q. 셀 병합은 언제 유용하고 언제 문제가 될까요?**
> **유용할 때**: 제목 행, 보고서의 헤더처럼 시각적으로 강조하고 싶을 때.
> **문제가 될 때**: 병합된 셀이 포함된 범위는 ①정렬 불가 ②필터 불가 ③Ctrl+D 자동 채우기 불가 ④표(Table) 변환 불가입니다. 데이터를 다루는 표 영역에서는 병합을 피하세요.

**Q. 행 100개를 한 번에 삽입하려면 어떻게 해야 할까요?**
> 행 번호를 클릭하여 **100개의 행을 먼저 선택**한 후 우클릭 → 삽입하면 됩니다. 선택한 행 수만큼 새 행이 삽입됩니다. 예: 3번 행에서 102번 행까지 드래그 선택 → 삽입

**Q. 틀 고정을 해제하려면 어떻게 할까요?**
> 보기 탭 → 틀 고정 → **틀 고정 취소**를 클릭하면 됩니다. (이미 틀 고정이 되어있을 때 메뉴가 "틀 고정 취소"로 바뀝니다)
