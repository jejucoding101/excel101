# 🎯 화두 풀 — SUMIF, COUNTIF 계열

> 아래 주제를 골라 직접 실습해보세요!  
> 조건부 합계와 개수로 데이터를 분석합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 동아리 회비 관리 대장 ⭐ 추천
> 동아리 회원들의 회비 납부/지출 내역을 관리하고 집계하세요.
- SUMIF로 납부 합계 / 지출 합계
- COUNTIF로 납부 인원 / 지출 건수
- 개인별 납부 현황 (SUMIFS)

### 2. 카테고리별 가계부 분석
> 한 달 가계부 데이터(30건 이상)를 카테고리별로 분석하세요.
- 카테고리: 식비, 교통, 문화, 의류, 기타
- 각 카테고리별 SUMIF → 지출 합계
- 각 카테고리별 COUNTIF → 지출 횟수
- 각 카테고리별 AVERAGEIF → 평균 지출

---

## 🟡 도전 (조금 더 생각 필요)

### 3. 매출 보고서 분석
> 지역별·제품별 매출 데이터를 SUMIFS/COUNTIFS로 교차 분석하세요.
- 서울 + 노트북 매출 합계 (SUMIFS)
- 부산 + 판매 건수 (COUNTIFS)
- 지역별 평균 주문 금액 (AVERAGEIFS)

### 4. 학급 통계 대시보드
> 학생 30명의 데이터에서 다양한 통계를 추출하세요.
- 성별별 인원수 (COUNTIF)
- 성별별 평균 점수 (AVERAGEIF)
- 80점 이상 학생 수 (COUNTIF ">=" 조건)
- 남학생 중 90점 이상 (COUNTIFS)

---

## 🔴 심화 (탐구해보기)

### 5. 와일드카드 활용
> SUMIF/COUNTIF에서 와일드카드(*, ?)를 활용해보세요.
- `=COUNTIF(A:A, "*김*")` → 이름에 "김"이 포함된 사람 수
- `=COUNTIF(A:A, "김??")` → "김"으로 시작하는 3글자 이름
- `=SUMIF(A:A, "A*", B:B)` → A로 시작하는 제품의 합계
