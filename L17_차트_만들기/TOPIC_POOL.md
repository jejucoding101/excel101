# 🎯 화두 풀 — 차트 만들기

> 아래 주제를 골라 직접 실습해보세요!

---

## 🟢 입문

### 1. 월별 매출 추이 차트 ⭐ 추천
> 12개월 매출 데이터로 막대+꺾은선 차트를 만들고, 추세를 분석하세요.

### 2. 설문 결과 시각화
> 설문조사 결과(선호도, 만족도 등)를 원형 차트로 비율을 표시하세요.

---

## 🟡 도전

### 3. 복합 차트 만들기
> 매출(막대)과 이익률(꺾은선)을 하나의 차트에 보조 축으로 표시하세요.
- 삽입 → 추천 차트 → 콤보(복합)

### 4. 과목별 성적 레이더 차트
> 5과목 점수를 방사형(레이더) 차트로 시각화하세요. 균형 잡힌 학생은 누구?

---

## 🔴 심화

### 5. 인터랙티브 차트 실험
> 드롭다운(16강)에서 월을 선택하면 해당 월 데이터 차트가 바뀌는 시스템을 만들어보세요.
- 힌트: VLOOKUP + 차트 데이터 소스를 수식으로 연결
