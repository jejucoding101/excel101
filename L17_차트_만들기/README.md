# 📌 17강: 데이터를 그림으로 — 차트 만들기

> **핵심 포인트**: 세로막대, 꺾은선, 원형 차트를 만들고, 차트 요소를 편집하며, 스파크라인으로 미니 차트를 삽입합니다.

---

## 📖 이론 (20분)

### 왜 차트인가?

```
숫자만 보면:                    차트로 보면:
서울 45,000                     서울 ████████████████ 45,000
부산 32,000                     부산 ███████████░░░░░ 32,000
대구 18,000                     대구 ██████░░░░░░░░░░ 18,000
인천 28,000                     인천 █████████░░░░░░░ 28,000

→ 숫자를 하나하나 비교해야 함    → 한눈에 크기 비교 가능!
```

> 💡 **"좋은 차트 하나가 숫자 100개보다 강력합니다"**

### 차트의 기본 요소

```
┌──────────────────────────────────────┐
│  📊 월별 매출 현황  ← 차트 제목      │
│                                      │
│  50 ─┤                               │
│  40 ─┤    ██                         │
│  30 ─┤    ██  ██                     │ ← Y축 (값 축)
│  20 ─┤ ██ ██  ██  ██                 │
│  10 ─┤ ██ ██  ██  ██  ██             │
│   0 ─┴──┬──┬──┬──┬──┬──             │
│         1월 2월 3월 4월 5월           │ ← X축 (항목 축)
│                                      │
│  ■ 매출(만원)  ← 범례               │
└──────────────────────────────────────┘
```

| 요소 | 설명 |
|------|------|
| **차트 제목** | 차트가 무엇을 보여주는지 |
| **X축 (가로축)** | 항목/카테고리 (월, 제품명 등) |
| **Y축 (세로축)** | 숫자 값 (매출, 점수 등) |
| **범례** | 데이터 시리즈 이름 |
| **데이터 레이블** | 각 막대/점 위에 숫자 표시 |
| **눈금선** | 값을 읽기 쉽게 하는 가로/세로 선 |

### 차트 종류 선택 가이드

| 차트 종류 | 아이콘 | 적합한 상황 | 예시 |
|----------|--------|-----------|------|
| **세로 막대** | 📊 | 항목 간 크기 비교 | 지역별 매출, 과목별 점수 |
| **가로 막대** | 📊↔ | 항목이 많을 때 | 20개 제품 비교 |
| **꺾은선** | 📈 | 시간에 따른 변화 추이 | 월별 매출 추이, 주가 변동 |
| **원형(파이)** | 🥧 | 전체 중 비율/구성 | 카테고리별 지출 비율 |
| **영역** | 📉 | 누적 추이 | 월별 누적 매출 |
| **분산형** | ⚬⚬ | 두 변수 간 관계 | 키 vs 몸무게 |
| **복합** | 📊📈 | 서로 다른 단위 비교 | 매출(막대) + 이익률(선) |

> 🔑 **차트 선택 핵심 규칙**:
> - **비교** → 막대 차트
> - **추이/변화** → 꺾은선 차트
> - **비율/구성** → 원형 차트

### 차트 만들기 3단계

```
1단계: 데이터 선택 (제목 행 포함!)
2단계: 삽입 → 원하는 차트 유형 클릭
3단계: 차트 요소 편집 (제목, 레이블 등)
```

### 추천 차트 기능

데이터를 선택 후 **삽입 → 추천 차트**를 클릭하면, 엑셀이 데이터에 맞는 차트를 자동으로 추천합니다!

### 스파크라인 — 셀 안의 미니 차트

셀 하나에 들어가는 아주 작은 차트입니다.

```
     A      B    C    D    E     F(스파크라인)
1행  제품   1월  2월  3월  4월   추이
2행  노트북  50   60   45   70   📈 (작은 꺾은선)
3행  마우스  30   35   40   38   📈 (작은 꺾은선)
```

삽입 → 스파크라인 → 꺾은선/세로막대/승패

### ⌨️ 이번 강의 필수 단축키

| 단축키 | 기능 |
|--------|------|
| `Alt+F1` | 현재 데이터로 차트 즉시 생성 |
| `F11` | 새 시트에 차트 생성 |

## 🔨 가이드 실습 (25분)

### 실습 1: 월별 매출 차트 3종 세트 (12분)

**목표**: 같은 데이터로 3가지 차트를 만들어 비교합니다.

1. **데이터 입력**:
   ```
        A      B
   1행  월     매출(만원)
   2행  1월    3200
   3행  2월    2800
   4행  3월    3500
   5행  4월    4100
   6행  5월    3800
   7행  6월    4500
   ```

2. **세로 막대 차트**: A1:B7 선택 → 삽입 → 세로 막대형 → 2D 세로 막대
   - 차트 제목 더블클릭 → "월별 매출 현황"으로 변경
   - `+` 버튼 → 데이터 레이블 체크 (막대 위에 숫자 표시)

3. **꺾은선 차트**: 같은 데이터 선택 → 삽입 → 꺾은선형
   - 추이(추세)가 한눈에 보이나요?

4. **원형 차트**: 같은 데이터 선택 → 삽입 → 원형
   - 데이터 레이블 → "백분율" 표시
   - 어느 달이 전체 매출에서 가장 큰 비중인지 한눈에!

5. **비교**: 3가지 차트 중 이 데이터에 가장 적합한 것은?

### 실습 2: 차트 요소 편집 (8분)

**목표**: 차트를 전문가 수준으로 꾸밉니다.

1. 막대 차트 선택 → 우측의 **+** 버튼:
   - ☑ 차트 제목 → 편집
   - ☑ 데이터 레이블 → 위쪽 표시
   - ☑ 축 제목 → Y축 "금액(만원)" 입력

2. **색상 변경**: 차트 선택 → 🎨 (스타일) 버튼 → 원하는 색상 테마

3. **특정 막대만 강조**: 가장 큰 막대를 더블클릭 → 채우기 색상을 별도로 변경

### 실습 3: 스파크라인 삽입 (5분)

1. 위 데이터 옆에 스파크라인 열 추가
2. 삽입 → 스파크라인 → 꺾은선 → 데이터 범위: B2:B7, 위치: C2
3. 셀 안에 미니 차트가 나타남!
