# 📌 9강: 기본 함수 마스터 — SUM, AVERAGE, COUNT

> **핵심 포인트**: 엑셀에서 가장 많이 쓰는 기본 함수 6가지(SUM, AVERAGE, COUNT, COUNTA, MAX, MIN)를 마스터합니다.

---

## 📖 이론 (20분)

### 함수(Function)란?

**미리 만들어져 있는 수식**입니다. 직접 수식을 쓰는 것보다 훨씬 편리합니다.

```
직접 수식으로 합계:              함수로 합계:
=A1+A2+A3+A4+A5+A6+...+A100    =SUM(A1:A100)
→ 100개를 하나하나 더해야 함!    → 한 줄로 끝!
```

### 함수의 기본 구조

```
=함수이름(인수1, 인수2, ...)

예시:
=SUM(A1:A10)
  │    └──── 인수(Argument): 함수에 전달하는 데이터
  └───────── 함수 이름
```

**인수 표기법**:
| 표기 | 의미 | 예시 |
|------|------|------|
| `A1:A10` | A1부터 A10까지의 **범위** | 연속된 셀들 |
| `A1,B1,C1` | 개별 셀들을 **쉼표로 구분** | 떨어진 셀들 |
| `A1:C10` | 2차원 범위 (직사각형) | 넓은 영역 |

### 6대 기본 함수

#### 1. SUM — 합계

```
=SUM(범위)
```

| 데이터 | 수식 | 결과 |
|--------|------|------|
| A1:A5 = 10, 20, 30, 40, 50 | `=SUM(A1:A5)` | **150** |

> 단축키: `Alt+=` → 자동으로 SUM 함수 삽입!

#### 2. AVERAGE — 평균

```
=AVERAGE(범위)
```

| 데이터 | 수식 | 결과 |
|--------|------|------|
| A1:A5 = 10, 20, 30, 40, 50 | `=AVERAGE(A1:A5)` | **30** |

> 💡 빈 셀은 평균 계산에서 **제외**됩니다. 0이 입력된 셀은 포함!

#### 3. COUNT — 숫자 개수

```
=COUNT(범위)
```

**숫자**가 들어있는 셀의 개수를 셉니다.

| A열 데이터 | 결과 |
|-----------|------|
| 10, 20, "abc", 30, (빈칸) | `=COUNT(A1:A5)` → **3** (숫자만 3개) |

#### 4. COUNTA — 비어있지 않은 셀 개수

```
=COUNTA(범위)
```

**뭔가 들어있는** 셀의 개수를 셉니다 (숫자든 문자든).

| A열 데이터 | 결과 |
|-----------|------|
| 10, 20, "abc", 30, (빈칸) | `=COUNTA(A1:A5)` → **4** (빈칸 빼고 4개) |

#### 5. MAX — 최대값

```
=MAX(범위)
```

| 데이터 | 수식 | 결과 |
|--------|------|------|
| 85, 92, 78, 95, 88 | `=MAX(A1:A5)` | **95** |

#### 6. MIN — 최소값

```
=MIN(범위)
```

| 데이터 | 수식 | 결과 |
|--------|------|------|
| 85, 92, 78, 95, 88 | `=MIN(A1:A5)` | **78** |

### 함수 비교 요약

| 함수 | 기능 | 결과 예시 (10,20,30,40,50) |
|------|------|--------------------------|
| `SUM` | 합계 | 150 |
| `AVERAGE` | 평균 | 30 |
| `COUNT` | 숫자 셀 개수 | 5 |
| `COUNTA` | 비어있지 않은 셀 | 5 |
| `MAX` | 최대값 | 50 |
| `MIN` | 최소값 | 10 |

### 함수 마법사 (Shift+F3)

함수가 기억나지 않을 때 **함수 마법사**를 활용하세요:

1. `Shift+F3` (또는 수식 입력줄 왼쪽의 **fx** 버튼)
2. 함수 이름 검색 또는 카테고리에서 선택
3. 인수 입력 도우미가 나타남 (어떤 값을 넣어야 하는지 안내)
4. 확인 → 수식 자동 완성

### 상태 표시줄의 숨은 기능

셀 범위를 선택하면 **화면 하단 상태 표시줄**에 자동으로 합계, 평균, 개수가 표시됩니다!

```
┌─────────────────────────────────────────────────────────────┐
│ 준비 완료          평균: 30   개수: 5   합계: 150            │
└─────────────────────────────────────────────────────────────┘
```

> 💡 수식을 안 써도 빠르게 확인할 수 있는 꿀팁! 셀 범위를 드래그만 하면 됩니다.

### ⌨️ 이번 강의 필수 단축키

| 단축키 | 기능 |
|--------|------|
| `Alt+=` | 자동 합계 (SUM) |
| `Shift+F3` | 함수 마법사 |

## 🔨 가이드 실습 (25분)

### 실습 1: 성적 요약표 만들기 (12분)

**목표**: 6가지 기본 함수를 모두 사용하여 용돈 기록을 분석합니다.

1. **데이터 입력**:
   ```
        A         B
   1행  날짜      금액
   2행  3/1       5000
   3행  3/3       12000
   4행  3/5       3000
   5행  3/8       8000
   6행  3/10      15000
   7행  3/12      2000
   8행  3/15      7000
   9행  3/18      25000
   10행 3/20      4000
   11행 3/25      9000
   ```

2. **함수 입력** (D열에 라벨, E열에 수식):
   ```
   D2: 총 지출     E2: =SUM(B2:B11)
   D3: 평균 지출   E3: =AVERAGE(B2:B11)
   D4: 지출 횟수   E4: =COUNT(B2:B11)
   D5: 최대 지출   E5: =MAX(B2:B11)
   D6: 최소 지출   E6: =MIN(B2:B11)
   ```

3. **서식 적용**: E열에 천 단위 쉼표(Ctrl+Shift+1), 제목 굵게

### 실습 2: 학생 성적 요약표 (10분)

**목표**: 과목별·학생별 합계와 평균을 함수로 계산합니다.

1. **데이터 입력**:
   ```
        A      B     C     D     E     F        G
   1행  이름   국어  수학  영어  과학  합계     평균
   2행  홍길동  85    92    78    88   (수식)   (수식)
   3행  김영희  90    75    95    82   (수식)   (수식)
   4행  이민수  72    88    84    90   (수식)   (수식)
   5행  박지은  95    91    87    76   (수식)   (수식)
   6행  정다빈  68    82    79    85   (수식)   (수식)
   7행
   8행  과목평균 (수식)(수식)(수식)(수식)
   9행  최고점  (수식)(수식)(수식)(수식)
   10행 최저점  (수식)(수식)(수식)(수식)
   ```

2. **학생별 합계/평균** (F2, G2):
   - F2: `=SUM(B2:E2)` → 채우기 핸들로 F6까지 복사
   - G2: `=AVERAGE(B2:E2)` → 채우기 핸들로 G6까지 복사

3. **과목별 통계** (8~10행):
   - B8: `=AVERAGE(B2:B6)` → 채우기 핸들로 E8까지 복사
   - B9: `=MAX(B2:B6)` → 채우기 핸들로 E9까지 복사
   - B10: `=MIN(B2:B6)` → 채우기 핸들로 E10까지 복사

### 실습 3: COUNT vs COUNTA 차이 체험 (5분)

**목표**: COUNT와 COUNTA의 차이를 명확히 이해합니다.

```
     A열 데이터       B열(COUNT)    C열(COUNTA)
1행  100              
2행  abc              
3행  200              
4행  (빈 셀)          
5행  300              
6행  가나다            
7행                   =COUNT(A1:A6) =COUNTA(A1:A6)
```

- COUNT 결과: 3 (숫자만: 100, 200, 300)
- COUNTA 결과: 5 (비어있지 않은 셀: 100, abc, 200, 300, 가나다)
