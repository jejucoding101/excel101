# 🎯 화두 풀 — 기본 함수 마스터

> 아래 주제를 골라 직접 실습해보세요!  
> SUM, AVERAGE, COUNT, MAX, MIN으로 데이터를 요약합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 월별 용돈 기록장 ⭐ 추천
> 한 달 동안의 수입/지출을 기록하고 합계, 평균, 최대/최소 지출을 분석하세요.
- 날짜, 내역, 수입, 지출 열 구성
- SUM → 총 수입, 총 지출
- AVERAGE → 일 평균 지출
- MAX/MIN → 가장 큰/작은 지출
- COUNT → 지출 발생 횟수

### 2. 체력 측정 결과표
> 학생 10명의 체력 측정(50m 달리기, 팔굽혀펴기, 윗몸일으키기, 앉아윗몸앞으로굽히기) 기록을 정리하세요.
- 각 종목별 평균, 최고기록, 최저기록 계산
- 학생별 합계 점수 계산

---

## 🟡 도전 (조금 더 생각 필요)

### 3. 운동 기록 분석기
> 한 달간 운동 기록(날짜, 종목, 시간(분), 소모칼로리)을 분석하세요.
- 총 운동 시간(SUM), 평균 운동 시간(AVERAGE)
- 최장 운동 시간(MAX), 운동 횟수(COUNT)
- COUNTA로 "운동한 날"과 "쉰 날" 개수 비교

### 4. 시험 성적 분석 보고서
> 학생 15명의 5과목 성적으로 종합 분석 보고서를 만드세요.
- 학생별 총점/평균, 과목별 평균/최고/최저
- 전체 학급 평균 계산
- 가장 높은 점수를 받은 과목 찾기

---

## 🔴 심화 (탐구해보기)

### 5. ROUND 함수 탐구
> 평균 등의 계산에서 소수점 처리를 연습해보세요.
- `=ROUND(3.456, 2)` → 3.46 (소수점 2자리)
- `=ROUND(3.456, 1)` → 3.5 (소수점 1자리)
- `=ROUND(3.456, 0)` → 3 (정수)
- `=ROUNDUP()` (올림)과 `=ROUNDDOWN()` (버림)도 시도해보세요

## 💬 생각해볼 점 — 해답

**Q. AVERAGE에서 빈 셀과 0이 입력된 셀의 처리 차이는?**
> - **빈 셀**: AVERAGE 계산에서 **제외**됩니다 (분모에 포함 안 됨)
> - **0이 입력된 셀**: **포함**됩니다 (분모에 포함, 평균이 낮아짐)
> 예: 10, 20, 빈칸 → 평균 15. / 10, 20, 0 → 평균 10.

**Q. 1000명의 점수 합계를 `+`로 일일이 더하면 얼마나 걸릴까요?**
> 사실상 불가능합니다. 수식이 너무 길어지고, 한 명이 빠지거나 추가되면 수식 전체를 수정해야 합니다. `=SUM(B2:B1001)` 한 줄이면 끝!

**Q. 상태 표시줄의 빠른 통계 기능을 일상 업무에서 어떻게 활용할 수 있을까요?**
> 수식을 입력하지 않고도 범위를 드래그하면 하단에 합계, 평균, 개수가 즉시 표시됩니다.
> 활용 사례: "이 행들의 합계가 맞나?" 빠른 검증, 보고서 작성 전 통계 확인, COUNT로 데이터 행 수 파악.
