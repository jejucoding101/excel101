# 📌 15강: VLOOKUP과 XLOOKUP — 데이터 자동 조회

> **핵심 포인트**: VLOOKUP으로 다른 표에서 데이터를 찾아오고, 최신 XLOOKUP의 편리함을 배웁니다.

---

## 📖 이론 (20분)

### VLOOKUP이란?

**V**ertical **LOOKUP** — 세로 방향으로 값을 찾아, **같은 행의 다른 열** 데이터를 가져옵니다.

```
=VLOOKUP(찾을값, 표범위, 열번호, [일치유형])

예시: 품목코드로 상품명 찾기

"제품 목록" 표:              주문서:
     A(코드) B(상품명) C(가격)       F(코드) G(상품명)
1행  A001   노트북   1500000       A003    =VLOOKUP(F1, A:C, 2, 0)
2행  A002   마우스   35000                  → "키보드"를 자동으로!
3행  A003   키보드   80000
4행  A004   모니터   450000
```

### VLOOKUP 4가지 인수

| 인수 | 설명 | 예시 |
|------|------|------|
| **찾을값** | 찾고자 하는 값 | `F1` (또는 "A003") |
| **표범위** | 찾을 데이터가 있는 전체 표 | `A1:C4` |
| **열번호** | 가져올 열의 번호 (1부터 시작) | `2` (B열=상품명) |
| **일치유형** | 0=정확히 일치, 1=근사치 | `0` (거의 항상 0!) |

> ⚠️ **최대 주의**: 일치유형은 **거의 항상 `0` (FALSE, 정확히 일치)**을 사용하세요! `1`은 오류의 원인이 됩니다.

### VLOOKUP의 한계

| 한계 | 설명 |
|------|------|
| 🚫 왼쪽 조회 불가 | 찾을값이 반드시 표범위의 **첫 번째 열**에 있어야 함 |
| 🚫 열 번호 하드코딩 | 열을 삽입/삭제하면 번호가 틀어짐 |
| 🚫 아래에서 위 검색 불가 | 항상 위에서 아래로만 검색 |

### XLOOKUP — VLOOKUP의 진화 ⭐

```
=XLOOKUP(찾을값, 찾는범위, 반환범위, [없을때값], [일치모드], [검색모드])

예시:
=XLOOKUP(F1, A1:A4, B1:B4, "없음")
              │       │       │
              │       │       └── 못 찾으면 "없음" 표시
              │       └────────── B열에서 가져와!
              └────────────────── A열에서 찾아!
```

### VLOOKUP vs XLOOKUP 비교

| 기능 | VLOOKUP | XLOOKUP |
|------|---------|---------|
| 왼쪽 조회 | ❌ 불가 | ✅ 가능 |
| 열 번호 | 숫자로 지정 | 반환 범위로 지정 |
| 오류 처리 | IFERROR 필요 | 내장 (`없을때값` 인수) |
| 기본 일치 | 근사치 (주의!) | 정확히 일치 (안전!) |
| 지원 버전 | 모든 버전 | Excel 365 / 2021+ |

### INDEX + MATCH — 전통적 대안
```
=INDEX(반환범위, MATCH(찾을값, 찾는범위, 0))

예시:
=INDEX(B1:B4, MATCH("A003", A1:A4, 0))
→ A열에서 "A003"의 위치(행번호)를 찾고, B열에서 그 위치의 값 반환
```

> 💡 여전히 많은 실무 환경에서 INDEX+MATCH를 사용합니다. XLOOKUP을 못 쓰는 환경에서 유용!

### IFERROR + VLOOKUP 조합

```
=IFERROR(VLOOKUP(F1, A:C, 2, 0), "찾을 수 없음")
→ 못 찾으면 #N/A 대신 "찾을 수 없음" 표시
```

## 🔨 가이드 실습 (25분)

### 실습 1: 재고 관리 시스템 (12분)

**목표**: 품목 코드를 입력하면 상품 정보가 자동으로 표시되는 시스템을 만듭니다.

1. **제품 마스터 시트 만들기** (시트명: "제품목록"):
   ```
        A         B        C        D
   1행  제품코드   제품명   단가     카테고리
   2행  P001      A4용지   25000    사무용품
   3행  P002      볼펜     3500     필기구
   4행  P003      포스트잇  1200    사무용품
   5행  P004      스테이플러 15000   사무용품
   6행  P005      수정테이프 2500    필기구
   ```

2. **주문서 시트 만들기** (시트명: "주문서"):
   ```
        A         B         C        D           E
   1행  제품코드   제품명    단가     수량        금액
   2행  P003      (수식)    (수식)   10         (수식)
   3행  P001      (수식)    (수식)   5          (수식)
   4행  P005      (수식)    (수식)   20         (수식)
   ```

3. **VLOOKUP 수식**:
   - B2: `=VLOOKUP(A2, 제품목록!A:D, 2, 0)` → 제품명 자동!
   - C2: `=VLOOKUP(A2, 제품목록!A:D, 3, 0)` → 단가 자동!
   - E2: `=C2*D2` → 금액

4. **IFERROR 추가** (안전하게): 
   - B2를 `=IFERROR(VLOOKUP(A2, 제품목록!A:D, 2, 0), "")` 로 수정

5. **XLOOKUP 버전** (Excel 365 사용자):
   - B2: `=XLOOKUP(A2, 제품목록!A:A, 제품목록!B:B, "")` → 더 직관적!


> ⚠️ **막힐 수 있는 포인트**:
> - **VLOOKUP 4번째 인수(0)를 빼먹었어요** → 4번째 인수가 없거나 TRUE(1)이면 "근사값 검색"이 되어 잘못된 결과가 나올 수 있습니다. 정확한 검색에는 반드시 `0`(또는 FALSE)을 넣으세요
> - **`#N/A` 오류가 나요** → 검색값(제품코드)이 제품 목록의 A열에 정확히 일치하는 값이 있는지 확인하세요. 대소문자, 공백 차이도 오류의 원인이 됩니다
> - **다른 시트를 참조하는 법을 모르겠어요** → `시트이름!셀범위` 형식입니다: `제품목록!A:D`. 시트 이름에 공백이 있으면 작은따옴표로 감쌉니다: `'시트 이름'!A:D`

### 실습 2: VLOOKUP vs XLOOKUP 비교 (8분)

**목표**: 같은 검색을 두 함수로 작성하고, 차이를 체험합니다.

1. 실습 1의 주문서에서 **G열**을 추가하여 XLOOKUP 버전을 작성하세요
   - G2: `=XLOOKUP(A2, 제품목록!A:A, 제품목록!B:B, "미등록")`
2. B열(VLOOKUP)과 G열(XLOOKUP)의 결과가 동일한지 확인
3. 차이를 정리: VLOOKUP은 열 번호가 필요하고, XLOOKUP은 반환 열을 직접 지정


> ⚠️ **막힐 수 있는 포인트**:
> - **XLOOKUP 함수가 인식이 안 돼요** → XLOOKUP은 Excel 365 또는 Excel 2021 이상에서만 사용 가능합니다. 이전 버전이라면 VLOOKUP으로 대체하세요
> - **VLOOKUP의 열 번호(col_index)가 헷갈려요** → 검색 열(A열)부터 순서대로 세면 됩니다: A=1, B=2, C=3, D=4. 제품명은 2, 단가는 3입니다

### 실습 3: 오류 처리 연습 (5분)

**목표**: 존재하지 않는 값을 검색할 때 오류를 처리하는 방법을 익힙니다.

1. 주문서 A열에 존재하지 않는 코드 `P999`를 입력하세요
2. IFERROR 없는 수식의 결과를 확인하세요 → `#N/A` 오류!
3. 수식을 `=IFERROR(VLOOKUP(...), "미등록 제품")`으로 수정하세요
4. 다시 확인 → "미등록 제품"이 표시되나요?


> ⚠️ **막힐 수 있는 포인트**:
> - **IFERROR로 감싸도 여전히 오류가 보여요** → IFERROR의 첫 번째 인수에 전체 VLOOKUP 수식이 들어가야 합니다: `=IFERROR(VLOOKUP(A2,...), "미등록 제품")`
> - **"미등록 제품"이라는 텍스트 대신 빈 셀로 두고 싶어요** → 두 번째 인수를 `""`(빈 문자열)로 지정하세요: `=IFERROR(VLOOKUP(A2,...), "")`
