# 🎯 화두 풀 — VLOOKUP과 XLOOKUP

> 아래 주제를 골라 직접 실습해보세요!

---

## 🟢 입문

### 1. 재고 관리 시스템 ⭐ 추천
> 품목 코드를 입력하면 제품명, 단가, 카테고리가 자동으로 표시되는 주문서를 만드세요.

### 2. 성적 등급 조회표
> 점수 범위에 따른 등급표를 만들고, VLOOKUP 근사치 일치(1)로 등급을 자동 배정하세요.

---

## 🟡 도전

### 3. 직원 정보 조회 시스템
> 사번을 입력하면 이름, 부서, 직급, 연락처가 자동으로 나타나는 시스템을 만드세요.
- VLOOKUP으로 각 열 번호(2,3,4,5)를 다르게 적용

### 4. 우편번호 → 지역명 변환
> 우편번호 목록에서 VLOOKUP/XLOOKUP으로 지역명을 자동 매칭하세요.
