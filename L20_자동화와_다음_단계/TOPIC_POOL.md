## 🟢 입문

### 1. Flash Fill 연습 ⭐ 추천
> 다양한 데이터(이름, 전화번호, 이메일, 주소)로 Flash Fill(Ctrl+E)의 패턴 인식 능력을 시험하세요.
- 어떤 패턴은 잘 인식하고, 어떤 패턴은 못 할까요?
- LEFT/MID/RIGHT 수식 vs Flash Fill, 어느 것이 더 편한지 비교!

### 2. 나만의 서식 매크로 만들기
> 자주 사용하는 서식(제목 꾸미기, 테두리, 배경색)을 매크로로 기록하세요.
- 기록 → 테스트 → .xlsm 저장

---

## 🟡 도전

### 3. 반복 업무 자동화 체험 🔥
> 실제로 매주 반복하는 업무를 하나 골라 자동화해보세요.
- 파워 쿼리로 데이터 정리?
- 매크로로 서식 적용?
- 피벗 테이블로 요약?

### 4. 파워 쿼리 탐험
> 데이터 → 데이터 가져오기 → "파일에서" → CSV 파일을 불러와 정리해보세요.
- 불필요한 열 제거, 필터 적용, 데이터 형식 변환
- "닫기 및 로드"로 결과 확인

## 🔴 심화

### 5. 종합 프로젝트: 나만의 관리 시스템
> 20강 동안 배운 모든 기능을 총동원하여 하나의 관리 시스템을 만들어보세요!
- 예: 개인 재테크 관리, 소규모 매장 관리, 동아리 관리
- 필수 요소: 데이터 입력(유효성 검사), 자동 계산(수식/함수), 요약(피벗), 시각화(차트), 대시보드
