### 1. Flash Fill — 패턴 자동 인식 ⭐

엑셀이 입력 패턴을 **자동으로 인식**하여 나머지를 채워주는 마법 같은 기능!

```
A열 (원본)           B열 (원하는 결과)
홍길동               홍  ← 첫 글자만 입력하면
김영희               김  ← Ctrl+E 누르면
이민수               이  ← 나머지 자동 완성!
박지은               박
```

**사용법**: 원하는 패턴을 1~2개 입력 → `Ctrl+E` (Flash Fill)

```
놀라운 활용 예시:

원본: "홍길동(남,25세)"
→ 이름만: "홍길동"         (패턴 1개 입력 → Ctrl+E)
→ 성별만: "남"             (패턴 1개 입력 → Ctrl+E)
→ 나이만: "25"             (패턴 1개 입력 → Ctrl+E)
```

> 🎵 수식 없이 텍스트를 쪼개고, 합치고, 변환할 수 있습니다!

### 2. 파워 쿼리 (Power Query) — 데이터 전처리 자동화

반복적인 데이터 정리 작업을 **한 번 기록해두면 버튼 하나로 반복**할 수 있습니다.

```
매주 받는 보고서:           파워 쿼리 적용 후:
┌──────────────────┐       ┌──────────────────┐
│ 지저분한 원본     │       │ 깔끔한 결과       │
│ - 빈 행 있음     │  →     │ - 빈 행 제거됨    │
│ - 열 이름 다름   │ 클릭!  │ - 열 이름 통일    │
│ - 날짜 형식 다름 │       │ - 날짜 형식 통일   │
└──────────────────┘       └──────────────────┘
```

**체험하기**:
1. 데이터 → **데이터 가져오기** → "표/범위에서"
2. 파워 쿼리 편집기가 열림
3. 여기서 "열 제거", "필터", "데이터 형식 변환" 등 작업
4. "닫기 및 로드" → 결과가 엑셀로!

> 💡 한 번 설정하면 나중에 **"새로 고침"** 버튼만 누르면 같은 작업이 자동 반복!

### 3. 매크로 (VBA) — 엑셀 자동화의 끝판왕

반복적인 **클릭 작업을 기록**하여 버튼 하나로 실행합니다.

```
매크로 기록 = 녹음 버튼 🎙️

1. 보기 → 매크로 → 매크로 기록 시작
2. 원하는 작업 수행 (서식, 정렬, 복사 등)
3. 매크로 기록 중지
4. 다음부터는 버튼 클릭만으로 같은 작업 반복!
```

**간단 체험**:
1. 보기 → **매크로** → 매크로 기록 → 이름: "서식정리" → 확인
2. 1행 선택 → 배경색 파랑 → 글꼴 흰색 → 굵게 → 열 너비 자동 맞춤
3. 보기 → 매크로 → **기록 중지**
4. 다른 시트에서 보기 → 매크로 → "서식정리" 실행 → 같은 서식이 자동 적용!

> ⚠️ 매크로가 있는 파일은 `.xlsm` 형식으로 저장해야 합니다!

## 🔨 가이드 실습 (25분)

### 실습 1: Flash Fill 마법 체험 (8분)

1. **이름 분리**:
   ```
   A열: "홍길동(남)"    B열: (B1에 "홍길동" 입력 → Ctrl+E)
        "김영희(여)"         → "김영희" 자동!
        "이민수(남)"         → "이민수" 자동!
   ```

2. **이메일 생성**:
   ```
   A열: "홍길동"    B열: (B1에 "gildong@company.com" 입력 → Ctrl+E)
        "김영희"         → 패턴 인식 시도
   ```

3. **전화번호 포맷 변경**:
   ```
   A열: "01012345678"    B열: (B1에 "010-1234-5678" 입력 → Ctrl+E)
        "01098765432"         → "010-9876-5432" 자동!
   ```

### 실습 2: 매크로 첫 체험 (10분)

1. **간단한 서식 매크로 기록**:
   - 보기 → 매크로 → 매크로 기록 → 이름: "표꾸미기"
   - 1행 선택 → 배경색 진한 파랑 → 글꼴 흰색 → 굵게
   - 테두리→ 모든 테두리
   - 보기 → 매크로 → 기록 중지

2. **새 시트에서 테스트**:
   - 새 시트에 아무 데이터 입력
   - 보기 → 매크로 → "표꾸미기" → 실행
   - 같은 서식이 자동 적용! 🎉

3. **`.xlsm`으로 저장**: F12 → "Excel 매크로 사용 통합 문서"로 저장
