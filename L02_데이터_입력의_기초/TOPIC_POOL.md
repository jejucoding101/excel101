# 🎯 화두 풀 — 데이터 입력의 기초

> 아래 주제를 골라 직접 실습해보세요!  
> 자동 채우기와 다양한 데이터 타입 입력을 활용합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 1년 달력 자동 생성 ⭐ 추천
> 자동 채우기를 활용하여 2025년 12개월 달력을 만들어보세요.
- Sheet1에 1월, Sheet2에 2월, ... 이런 식으로 시트를 나눠도 좋아요
- 1행에 "일~토" 요일을 자동 채우기로 입력
- 각 월의 날짜를 채워 넣어보세요
- **도전**: 날짜 시작 요일이 맞게 빈칸도 배치해보세요

### 2. 나만의 단어장
> 영어(또는 다른 외국어) 단어장을 만들어보세요.
- A열: 번호(자동 채우기), B열: 영어 단어, C열: 한국어 뜻, D열: 예문
- 20개 이상 입력해보세요

### 3. 좋아하는 음식 TOP 20
> 좋아하는 음식 순위표를 만들어보세요.
- A열: 순위(1~20), B열: 음식 이름, C열: 종류(한식/중식/일식/양식), D열: 평균 가격

---

## 🟡 도전 (조금 더 생각 필요)

### 4. 한 달 날씨 기록표
> 이번 달 또는 지난달의 날씨를 기록하는 표를 만들어보세요.
- A열: 날짜(자동 채우기!), B열: 요일(자동 채우기!), C열: 날씨, D열: 최고기온, E열: 최저기온
- 날짜와 요일은 자동 채우기로 한 번에!
- 기온은 숫자로 입력 (소수점도 OK)

### 5. 우리 반 생일 달력
> 반 친구들의 생일을 월별로 정리하는 표를 만들어보세요.
- A열: 이름, B열: 생일(날짜 형식), C열: 생일 달(1월~12월)
- 날짜 형식으로 입력하면 나중에 정렬할 수 있어요!

---

## 🔴 심화 (탐구해보기)

### 6. 데이터 타입 실험실
> 다양한 값을 입력하고 엑셀이 어떻게 해석하는지 실험해보세요.
- `100원`, `100`, `'100` — 각각 어떻게 다른가요?
- `1/2` — 숫자 0.5? 날짜 1월 2일? 문자 1/2?
- `TRUE`, `FALSE` — 이건 뭘까요?
- `=1+1` — 앞에 `=`을 붙이면 어떻게 되나요? (다음 강의 미리보기!)

## 💡 생각해볼 점

- 자동 채우기가 안 되는 데이터도 있을까요? 어떤 경우?
- 셀 하나에 입력할 수 있는 글자 수에 제한이 있을까요?
- 같은 숫자 `100`이라도 "문자 100"과 "숫자 100"은 무엇이 다를까요?
