# 🎯 화두 풀 — 데이터 입력의 기초

> 아래 주제를 골라 직접 실습해보세요!
> 자동 채우기와 다양한 데이터 타입 입력을 여행사 실무에 활용합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 고객 예약 데이터베이스 ⭐ 추천
> 여행사에서 실제로 사용할 수 있는 고객 예약 대장을 만들어보세요.
- A열: 예약번호(TK-2501, TK-2502... 자동 채우기로!), B열: 고객명, C열: 연락처, D열: 출발일, E열: 상품명
- 연락처 입력 시 앞의 0이 사라지지 않도록 처리하세요 (`'010...` 또는 텍스트 형식 적용)
- 출발일은 날짜 형식으로 입력 (2025-05-01)
- 최소 10명 이상 입력

### 2. 여행 상품 목록표
> 판매 중인 여행 상품 목록을 정리해보세요.
- A열: 번호(자동 채우기), B열: 상품명, C열: 목적지, D열: 출발일, E열: 박수, F열: 가격
- 숫자(가격, 박수)와 날짜(출발일)와 문자(상품명)가 어떻게 다르게 표시되는지 확인!
- 20개 이상 입력

### 3. 월별 출발 일정 자동 생성
> 한 달 동안 매주 화·목 출발하는 일정표를 자동 채우기로 만들어보세요.
- A열: 출발일(날짜 자동 채우기), B열: 상품명, C열: 정원, D열: 예약 인원
- 출발 요일이 맞게 날짜를 입력 후 채우기 핸들 활용

---

## 🟡 도전 (조금 더 생각 필요)

### 4. 고객 국적별 분포 기록표
> 이번 달 방문한 외국인 고객을 국적별로 집계하는 표를 만들어보세요.
- A열: 국가명, B열: 인원수, C열: 주요 상품, D열: 특이사항
- 국가명(문자)과 인원수(숫자)의 정렬 방향 차이를 확인!
- **도전**: 요일별 자동 채우기로 "월요일~일요일" 열을 추가

### 5. 가이드 스케줄표
> 소속 가이드의 이번 달 일정을 기록하는 표를 만들어보세요.
- A열: 날짜(자동 채우기로 한 달치!), B열: 요일(자동 채우기!), C열: 배정 가이드, D열: 투어 상품, E열: 출발 시간
- 날짜와 요일을 각각 자동 채우기로 채워보세요

---

## 🔴 심화 (탐구해보기)

### 6. 예약 코드 데이터 타입 실험
> 예약 데이터를 다양한 방식으로 입력하고 엑셀이 어떻게 해석하는지 실험해보세요.
- `TK2501` — 문자? 숫자?
- `2025-05-01` — 날짜로 인식되나요? 셀 형식을 확인해보세요
- `1500000` — 숫자 계산 가능, `150만원` — 계산 가능?
- `'01012345678` — 작은따옴표 없이 입력하면 0이 사라지나요?
- `=100+200` — `=` 을 앞에 붙이면 어떻게 되나요? (다음 강의 미리보기!)

## 💬 생각해볼 점 — 해답

**Q. 자동 채우기가 안 되는 데이터도 있을까요?**
> 네! 패턴이 없는 임의의 텍스트(예: "사과", "컴퓨터", "강아지")는 그대로 복사만 됩니다. 자동 채우기는 **연속 숫자, 요일, 월, 날짜처럼 규칙이 있는 데이터**에서만 순열이 자동 생성됩니다.

**Q. 셀 하나에 입력할 수 있는 글자 수에 제한이 있을까요?**
> 있습니다! 셀 하나에는 최대 **32,767자**까지 입력 가능합니다. 단, 화면에 표시되는 길이는 열 너비에 따라 제한됩니다.

**Q. 같은 숫자 `100`이라도 "문자 100"과 "숫자 100"은 무엇이 다를까요?**
> **숫자 100** → SUM, AVERAGE 등 계산 함수에 포함됩니다. 오른쪽 정렬.
> **문자 100** (`'100`으로 입력) → 계산 함수에서 무시됩니다. =SUM에서 세지 않아요! 왼쪽 정렬.
> 전화번호나 우편번호처럼 계산이 필요 없는 숫자는 문자로 저장해야 합니다.
