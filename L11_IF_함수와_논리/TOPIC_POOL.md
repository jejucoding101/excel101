# 🎯 화두 풀 — IF 함수와 논리 판단

> 아래 주제를 골라 직접 실습해보세요!  
> IF, AND, OR, 중첩 IF를 활용하여 조건부 판단 시스템을 만듭니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 시험 등급 자동 판정기 ⭐ 추천
> 점수를 입력하면 수/우/미/양/가 등급을 자동 매기는 시스템을 만드세요.
- 중첩 IF 또는 IFS 함수 활용
- 합격/불합격 열도 추가
- 조건부 서식으로 "가" 등급에 빨간 배경

### 2. 출석 상태 자동 판별
> B열에 도착 시각을 입력하면 C열에 "출석"/"지각"/"결석"을 자동 표시하세요.
- 09:00 이전 → 출석
- 09:00~09:30 → 지각
- 빈 셀 → 결석 (IFERROR 또는 IF+빈셀 비교)

---

## 🟡 도전 (조금 더 생각 필요)

### 3. 택배비 자동 계산기
> 무게(kg)에 따라 택배비가 달라지는 시스템을 만드세요.
- 1kg 이하: 3,000원
- 1~5kg: 5,000원
- 5~10kg: 7,000원
- 10kg 초과: 10,000원
- 중첩 IF로 구현

### 4. 보너스 지급 판정
> 직원의 실적이 목표를 달성했는지 판별하고, AND/OR로 보너스를 계산하세요.
- 매출 목표 달성 AND 출근율 95% 이상 → 보너스 100%
- 매출 목표 달성 OR 출근율 95% 이상 → 보너스 50%
- 둘 다 미달 → 보너스 없음
