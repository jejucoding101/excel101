# 🎯 화두 풀 — IF 함수와 논리 판단

> 아래 주제를 골라 직접 실습해보세요!  
> IF, AND, OR, 중첩 IF를 활용하여 조건부 판단 시스템을 만듭니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 시험 등급 자동 판정기 ⭐ 추천
> 점수를 입력하면 수/우/미/양/가 등급을 자동 매기는 시스템을 만드세요.
- 중첩 IF 또는 IFS 함수 활용
- 합격/불합격 열도 추가
- 조건부 서식으로 "가" 등급에 빨간 배경

### 2. 출석 상태 자동 판별
> B열에 도착 시각을 입력하면 C열에 "출석"/"지각"/"결석"을 자동 표시하세요.
- 09:00 이전 → 출석
- 09:00~09:30 → 지각
- 빈 셀 → 결석 (IFERROR 또는 IF+빈셀 비교)

### 3. 택배비 자동 계산기
> 무게(kg)에 따라 택배비가 달라지는 시스템을 만드세요.
- 1kg 이하: 3,000원
- 1~5kg: 5,000원
- 5~10kg: 7,000원
- 10kg 초과: 10,000원
- 중첩 IF로 구현

### 4. 보너스 지급 판정
> 직원의 실적이 목표를 달성했는지 판별하고, AND/OR로 보너스를 계산하세요.
- 매출 목표 달성 AND 출근율 95% 이상 → 보너스 100%
- 매출 목표 달성 OR 출근율 95% 이상 → 보너스 50%
- 둘 다 미달 → 보너스 없음

## 💬 생각해볼 점 — 해답

**Q. 중첩 IF에서 조건 순서를 바꾸면 결과가 달라질 수 있을까요?**
> **네, 달라집니다!** 중첩 IF는 조건을 위에서부터 순서대로 확인합니다.
> ❌ 잘못된 예: `=IF(B2>=60,"양",IF(B2>=80,"우",IF(B2>=90,"수","가")))`
> → 90점도 처음 B2>=60 조건에서 "양"이 됩니다!
> ✅ 올바른 순서: 큰 기준부터(`>=90` → `>=80` → ...) 작성해야 합니다.

**Q. IF 함수에서 참/거짓 값에 숫자를 넣으면 어떤 활용이 가능할까요?**
> 출석/결석을 0과 1로 변환하여 집계: `=IF(B2="출석",1,0)` → SUM으로 출석 횟수 계산
> 점수 초과분 계산: `=IF(A1>100, A1-100, 0)` → 초과분만 계산
> IF 결과를 다른 수식에 바로 활용할 수 있어 매우 강력합니다.

**Q. "없으면 빈칸" 처리는 `=IF(조건, 값, "")` 이렇게!**
> 맞습니다! `""` (큰따옴표 두 개)는 빈 문자열을 나타냅니다. 예: `=IF(A1>0, A1*1.1, "")` → 양수면 10% 더한 값, 아니면 빈 셀처럼 보입니다. 단, 실제로 빈 셀은 아니므로 COUNTA에서 세어집니다.
