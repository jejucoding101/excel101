# 🎯 화두 풀 — 셀 서식의 마법

> 아래 주제를 골라 직접 실습해보세요!  
> 글꼴, 배경색, 테두리, 표시 형식으로 전문가 수준의 문서를 만듭니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 견적서/청구서 양식 ⭐ 추천
> 실무에서 바로 쓸 수 있는 견적서를 만들어보세요.
- 회사명, 견적일자, 담당자 정보 영역
- 항목 표: 번호, 품명, 규격, 수량, 단가, 금액
- 금액은 통화 형식(₩)으로
- 제목 행은 배경색+굵게, 바깥쪽 굵은 테두리

### 2. 나만의 명함 디자인
> 셀 서식을 활용해 명함 느낌의 카드를 만들어보세요.
- 셀 병합으로 명함 크기(약 5열×3행) 영역 만들기
- 이름(큰 글씨, 굵게), 직함, 연락처, 이메일, 주소
- 배경색과 테두리로 디자인

### 3. 상품 가격표
> 좋아하는 카페 또는 식당 메뉴판을 엑셀로 만들어보세요.
- 메뉴명, 설명, 가격
- 가격은 통화 형식, 카테고리별 배경색 구분
- "HOT" 메뉴는 빨간 글씨로 강조

---

## 🟡 도전 (조금 더 생각 필요)

### 4. 월별 가계부 양식
> 한 달 치 가계부 양식을 전문가 수준으로 꾸며보세요.
- 날짜, 내역, 카테고리(식비/교통/문화 등), 수입, 지출
- 금액은 천 단위 쉼표
- 수입은 파란 글씨, 지출은 빨간 글씨
- 제목 영역에 해당 월과 이름 표시

### 5. 학급 성적표 양식
> 깔끔한 성적표 양식을 만들어보세요.
- 학생 이름, 국어, 수학, 영어, 과학, 사회, 평균
- 제목행과 학생명 열에 배경색 적용
- 평균은 소수점 1자리까지 표시
- 바깥쪽 굵은 테두리 + 안쪽 얇은 테두리

---

## 🔴 심화 (탐구해보기)

### 6. 사용자 지정 표시 형식
> Ctrl+1 → 표시 형식 → "사용자 지정"에서 직접 형식을 만들어보세요.
- `#,##0"원"` → 3,500,000원 (자동으로 "원" 붙이기)
- `0"점"` → 85점
- `yyyy"년" mm"월" dd"일"` → 2025년 03월 15일
- `[RED]#,##0;[BLUE]-#,##0` → 양수는 빨강, 음수는 파랑
