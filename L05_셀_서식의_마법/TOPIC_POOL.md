# 🎯 화두 풀 — 셀 서식의 마법

> 아래 주제를 골라 직접 실습해보세요!
> 글꼴, 배경색, 테두리, 표시 형식으로 여행사 실무 문서를 전문가답게 꾸밉니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 여행 상품 견적서 ⭐ 추천
> 고객에게 발송할 수 있는 전문적인 여행 견적서를 만들어보세요.
- 상단: 여행사명(큰 글씨, 굵게), 견적일자, 담당자명
- 항목 표: 상품명, 출발일, 박수, 인원, 1인 단가, 총 금액
- 금액은 **통화 형식(₩)** + 천 단위 쉼표
- 제목 행은 **배경색(진한 파랑)+흰 글씨+굵게**, 바깥쪽 굵은 테두리

### 2. 가이드 명함 디자인
> 여행사 가이드용 명함을 엑셀 셀 서식으로 만들어보세요.
- 셀 병합으로 명함 크기(약 5열×3행) 영역 만들기
- 가이드 이름(큰 글씨, 굵게), "현지 가이드 / Local Guide", 연락처, 이메일, 담당 지역
- 배경색과 테두리로 디자인 (여행사 브랜드 색상 적용)

### 3. 여행 상품 가격표
> 시즌별 상품 가격표를 엑셀로 만들어보세요.
- 상품명, 비수기 가격, 성수기 가격, 특가 가격
- 가격은 통화 형식
- 특가 상품은 **빨간 글씨**로 강조
- 카테고리별(동남아/일본/유럽) **배경색 구분**

### 4. 월별 매출 정산 대장
> 이번 달 매출과 비용을 기록하는 정산 대장를 전문가 수준으로 꾸며보세요.
- 날짜, 예약번호, 상품명, 수입, 지출, 잔액
- 금액은 **천 단위 쉼표**
- 수입은 **파란 글씨**, 지출은 **빨간 글씨**
- 제목 영역에 해당 월 표시

### 5. 고객 등급별 관리표
> VIP·일반·신규 고객을 색상으로 구분하는 관리표를 만들어보세요.
- 고객명, 누적 이용 횟수, 총 결제금액, 등급
- VIP(금색 배경), 일반(흰 배경), 신규(연두 배경) 등급별 색상 구분
- 바깥쪽 굵은 테두리 + 안쪽 얇은 테두리

## 🔴 심화 (탐구해보기)

### 6. 사용자 지정 표시 형식 — 여행사 실무 적용
> Ctrl+1 → 표시 형식 → "사용자 지정"에서 여행사 문서에 맞는 형식을 만들어보세요.
- `#,##0"원"` → 1,500,000원 (자동으로 "원" 붙이기)
- `0"박"` → 3박
- `yyyy"년" mm"월" dd"일"` → 2025년 05월 01일 (출발일 형식)
- `[RED]#,##0;[BLUE]-#,##0` → 수입은 빨강, 지출은 파랑

## 💬 생각해볼 점 — 해답

**Q. 표시 형식으로 "₩3,500,000"으로 바꾼 셀의 실제 값은?**
> **3500000** 그대로입니다. 표시 형식은 "어떻게 보여줄지"만 바꿀 뿐, 셀에 저장된 실제 값은 변하지 않습니다. 수식 입력줄에서 항상 실제 값을 확인할 수 있습니다.

**Q. 사용자 지정 형식 `#,##0"원"`과 직접 "3500000원"이라고 입력하는 것의 차이는?**
> **사용자 지정 형식**: 실제 값은 숫자(3500000)로 유지 → SUM 계산 가능
> **직접 입력 "3500000원"**: 문자열로 저장 → SUM, AVERAGE 등 계산에서 **무시됨**
> 계산이 필요한 금액은 반드시 숫자로 저장하고, 표시만 형식으로 바꿔야 합니다.

**Q. 전문가들이 표를 만들 때 2~3가지 색상만 사용하는 이유는?**
> 색이 많으면 **시각적 피로**가 생기고, 정작 중요한 정보가 눈에 띄지 않습니다. 색을 적게 쓸수록 강조하고 싶은 부분이 더 명확하게 보입니다. 디자인의 기본 원칙: "Less is more(적을수록 더 낫다)".
