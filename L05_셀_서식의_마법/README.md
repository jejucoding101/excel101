# 📌 5강: 셀 서식의 마법 — 밋밋한 표를 전문가 문서로

> **핵심 포인트**: 글꼴, 배경색, 테두리, 맞춤, 표시 형식을 활용하여 데이터를 보기 좋게 꾸밉니다.

---

## 📖 이론 (20분)

### 왜 서식이 중요한가?

같은 데이터라도 서식에 따라 **가독성**이 완전히 달라집니다:

```
서식 없는 표:                    서식 적용 후:
이름 나이 급여                   ┌─────────────────────────┐
홍길동 25 3500000                │  직원 급여 명세서        │
김철수 30 4200000                ├────┬────┬───────────────┤
이영희 28 3800000                │이름│나이│    급여       │
                                 ├────┼────┼───────────────┤
                                 │홍길동│ 25 │ ₩3,500,000  │
                                 │김철수│ 30 │ ₩4,200,000  │
                                 │이영희│ 28 │ ₩3,800,000  │
                                 └────┴────┴───────────────┘
```

### 셀 서식 대화상자 (`Ctrl+1`)

엑셀 서식의 **만능 도구**입니다. `Ctrl+1`을 누르면 6개 탭이 나타납니다:

| 탭 | 기능 | 이번 강의 학습 |
|----|------|---------------|
| **표시 형식** | 숫자/날짜/통화 등 표시 방법 | ✅ |
| **맞춤** | 정렬, 줄 바꿈, 각도 | ✅ |
| **글꼴** | 서체, 크기, 색상, 스타일 | ✅ |
| **테두리** | 셀 외곽선 | ✅ |
| **채우기** | 셀 배경색 | ✅ |
| **보호** | 셀 잠금 | 19강에서 학습 |

### 글꼴 서식

| 기능 | 리본 메뉴 위치 | 단축키 |
|------|--------------|--------|
| 글꼴 변경 | 홈 → 글꼴 드롭다운 | — |
| 크기 변경 | 홈 → 크기 드롭다운 | — |
| **굵게** | 홈 → B | `Ctrl+B` |
| *기울임* | 홈 → I | `Ctrl+I` |
| <u>밑줄</u> | 홈 → U | `Ctrl+U` |
| 글꼴 색 | 홈 → A▼ (글꼴 색) | — |

> 💡 **팁**: 제목 행은 **굵게(Bold) + 배경색**으로 데이터 행과 구분하세요!

### 맞춤 (Alignment)

| 가로 맞춤 | 설명 |
|----------|------|
| 왼쪽 맞춤 | 텍스트 기본값 |
| 가운데 맞춤 | 제목에 적합 |
| 오른쪽 맞춤 | 숫자 기본값 |

| 세로 맞춤 | 설명 |
|----------|------|
| 위쪽 | 셀 상단에 배치 |
| 가운데 | 셀 중앙에 배치 (가장 깔끔) |
| 아래쪽 | 셀 하단에 배치 (기본값) |

특수 맞춤 기능:

| 기능 | 설명 | 위치 |
|------|------|------|
| **줄 바꿈** | 셀 안에서 여러 줄 표시 | 홈 → 줄 바꿈 |
| **셀에 맞춤** | 텍스트 크기를 셀에 맞게 축소 | Ctrl+1 → 맞춤 |
| **텍스트 방향** | 글자를 세로나 대각선으로 | Ctrl+1 → 맞춤 → 각도 |

### 테두리 (Border)

셀에 외곽선을 추가합니다:

```
테두리 없음:     얇은 테두리:      굵은 바깥 테두리:
 A  B  C         ┌──┬──┬──┐       ┏━━┬──┬━━┓
 D  E  F         ├──┼──┼──┤       ┃D │E │F ┃
 G  H  I         ├──┼──┼──┤       ┃G │H │I ┃
                  └──┴──┴──┘       ┗━━┴──┴━━┛
```

- **홈 → 테두리 버튼** 옆 ▼ 클릭 → 다양한 테두리 옵션
- 자주 쓰는 것: "모든 테두리", "바깥쪽 굵은 테두리"

### 채우기 (배경색)

- **홈 → 페인트 통 아이콘** 옆 ▼ 클릭 → 색상 선택
- 제목 행에 진한 배경색 + 흰색 글씨 = 전문가 느낌!

### 표시 형식 ⭐ (가장 중요!)

셀에 저장된 **실제 값은 변하지 않고**, **보여지는 형태만** 바꿉니다.

| 카테고리 | 입력값 | 표시 결과 | 설명 |
|---------|--------|----------|------|
| 일반 | `3500000` | 3500000 | 기본값 |
| 숫자 | `3500000` | 3,500,000 | 천 단위 쉼표 |
| 통화 | `3500000` | ₩3,500,000 | 통화 기호 + 쉼표 |
| 백분율 | `0.85` | 85% | 100을 곱하고 % 표시 |
| 날짜 | `45678` | 2025-01-15 | 날짜 형식 |
| 시간 | `0.5` | 12:00:00 | 시간 형식 |

> 🔑 **핵심**: 표시 형식은 "화장"과 같습니다. 실제 데이터(피부)는 그대로이고, 보이는 모습만 바꿉니다!

#### 빠른 표시 형식 단축키

| 단축키 | 형식 |
|--------|------|
| `Ctrl+Shift+1` | 천 단위 쉼표 (숫자) |
| `Ctrl+Shift+4` | 통화 (₩) |
| `Ctrl+Shift+5` | 백분율 (%) |
| `Ctrl+Shift+3` | 날짜 |
| `Ctrl+Shift+~` | 일반 (서식 제거) |

## 🔨 가이드 실습 (25분)

### 실습 1: 견적서 만들기 (12분)

**목표**: 셀 서식을 활용하여 전문적인 견적서를 작성합니다.

1. **제목 영역 만들기**:
   - A1:F1 셀 병합 → `견 적 서` 입력
   - 글꼴 크기: 20pt, 굵게, 가운데 맞춤
   - 배경색: 진한 파랑, 글꼴 색: 흰색

2. **회사 정보**:
   ```
   A3: 견적일자    B3: 2025-03-15 (날짜 형식)
   A4: 업체명      B4: (주)가나다상사
   A5: 담당자      B5: 홍길동
   ```

3. **항목 표 만들기**:
   ```
        A         B        C         D          E           F
   7행  번호      품명     규격      수량       단가        금액
   8행  1        A4용지   1박스(5연)  10      25,000     250,000
   9행  2        볼펜     검정 12개입   5       3,500      17,500
   10행 3        포스트잇  76×76mm    20       1,200      24,000
   ```

4. **서식 적용**:
   - 7행(제목행): 배경색 연한 파랑, 굵게, 가운데 맞춤
   - E열,F열(금액): 통화 형식 (`Ctrl+Shift+4`) 또는 천 단위 쉼표
   - D열(수량): 가운데 맞춤
   - 모든 테두리 적용 (7~10행)
   - F열 아래에 "합계" 행 추가

5. **마무리**:
   - F11에 `합계` 라벨, 굵게
   - 전체적으로 열 너비 조절
   - `Ctrl+S`로 저장

### 실습 2: 표시 형식 체험하기 (8분)

**목표**: 같은 숫자를 다양한 표시 형식으로 변환해봅니다.

```
     A열(입력값)       B열(형식 이름)    C열(A열을 이 형식으로!)
1행  3500000           일반              3500000
2행  3500000           천 단위 쉼표      3,500,000
3행  3500000           통화              ₩3,500,000
4행  0.856             백분율            85.6%
5행  45678             날짜              2025-01-15
6행  0.7               백분율            70%
```

- A열에 같은 값을 입력
- C열에 A열 값을 복사 후 표시 형식만 변경
- **관찰**: C열의 실제 값(수식 입력줄에서 확인)은 변하지 않음을 확인!

### 실습 3: 색상 조합 연습 (5분)

**목표**: 보기 좋은 색상 조합을 찾아 표를 꾸밉니다.

전문가들이 자주 쓰는 색상 조합:

| 스타일 | 제목행 배경 | 제목행 글꼴 | 짝수행 배경 |
|--------|-----------|-----------|-----------|
| 비즈니스 블루 | 진한 파랑 | 흰색 | 아주 연한 파랑 |
| 모던 그레이 | 진한 회색 | 흰색 | 연한 회색 |
| 그린 프레시 | 진한 초록 | 흰색 | 연한 초록 |

아무 표를 만들어 3가지 스타일을 적용해보고, 가장 마음에 드는 것을 찾아보세요!
