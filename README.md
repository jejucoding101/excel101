## 🛠️ 시작하기 전에

### 필요한 것
1. **컴퓨터** — Windows 또는 Mac
2. **Microsoft Excel** — Microsoft 365 또는 Excel 2021 권장
3. **마우스** — 엑셀은 마우스와 키보드를 함께 사용합니다
4. **호기심** — 가장 중요! 🔥

### 강의 활용법

```
📖 이론 (20분)      → README.md를 읽으며 개념 이해
🔨 가이드 실습 (25분) → README.md의 실습 섹션을 따라하기  
🎯 자율 실습 (25분)   → TOPIC_POOL.md에서 주제를 골라 자유롭게 도전
```

### 각 강의 폴더 구조

```
LXX_강의제목/
├── README.md       ← 강의 본문 (이론 + 실습 가이드)
├── TOPIC_POOL.md   ← 🎯 자율 실습 주제 모음
└── images/         ← 참고 이미지/다이어그램
```

## ⌨️ 엑셀 핵심 단축키 모음

가장 자주 쓰는 단축키를 미리 알아두세요:

| 단축키 | 기능 | 중요도 |
|--------|------|--------|
| `Ctrl+S` | 저장 | ⭐⭐⭐ |
| `Ctrl+Z` | 되돌리기 | ⭐⭐⭐ |
| `Ctrl+C` / `Ctrl+V` | 복사 / 붙여넣기 | ⭐⭐⭐ |
| `Ctrl+X` | 잘라내기 | ⭐⭐ |
| `Ctrl+1` | 셀 서식 대화상자 | ⭐⭐⭐ |
| `F2` | 셀 내용 편집 | ⭐⭐⭐ |
| `F4` | 참조 전환 ($) | ⭐⭐⭐ |
| `Alt+=` | 자동 합계 (SUM) | ⭐⭐ |
| `Ctrl+방향키` | 데이터 끝으로 이동 | ⭐⭐ |
| `Ctrl+Shift+방향키` | 데이터 끝까지 선택 | ⭐⭐ |

> 💡 모든 단축키를 한 번에 외울 필요 없습니다. 각 강의에서 해당 회차에 필요한 단축키만 하나씩 익혀가세요!
