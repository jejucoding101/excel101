# 🎯 화두 풀 — 첫 수식 만들기

> 아래 주제를 골라 직접 실습해보세요!  
> 사칙연산과 자동 합계로 실생활 계산을 자동화합니다.

---

## 🟢 입문 (바로 시도 가능)

### 1. 장보기 자동 합계 계산기 ⭐ 추천
> 품목, 단가, 수량을 입력하면 금액(단가×수량)과 총합계가 자동 계산되는 표를 만드세요.
- 품목 10개 이상 입력
- 금액 = 단가 × 수량 (수식)
- 맨 아래에 SUM으로 총합계
- 단가나 수량을 바꿔보고 자동 갱신 확인!

### 2. 점수 합계/평균 계산기
> 학생 5명의 5과목 점수를 입력하고, 각 학생별 합계와 평균 열을 추가하세요.
- 합계 = 5과목 점수를 더하는 수식
- 평균 = 합계 / 5
- 채우기 핸들로 전체 학생에 수식 복사

### 3. 용돈 잔액 계산기
> 용돈 수입과 지출을 기록하고 잔액을 자동 계산하세요.
- A열: 날짜, B열: 내용, C열: 수입, D열: 지출, E열: 잔액
- 잔액 = 이전 잔액 + 수입 - 지출 (수식)

---

## 🟡 도전 (조금 더 생각 필요)

### 4. 더치페이 계산기
> 여러 명이 먹은 음식의 총액을 인원수로 나누는 계산기를 만드세요.
- 각 메뉴와 금액 입력
- 총합계 자동 계산
- 인원수 입력 셀
- 1인당 금액 = 총합계 / 인원수 (수식)
- 팁 포함 금액 = 1인당 금액 × 1.1 (수식)

### 5. BMI 계산기
> 키(m)와 몸무게(kg)를 입력하면 BMI를 자동 계산하세요.
- BMI = 몸무게 / (키 × 키), 즉 `=B2/(A2^2)` 또는 `=B2/(A2*A2)`
- 여러 사람의 데이터를 입력하여 비교

---

## 🔴 심화 (탐구해보기)

### 6. 복리 이자 계산기
> 원금, 이자율, 기간을 입력하면 매년 원리금을 계산하세요.
- A열: 연차(1~10), B열: 원리금
- B2 = 원금 × (1 + 이자율) (첫 해)
- B3 = B2 × (1 + 이자율) (셀 참조로!)
- 채우기 핸들로 10년치 계산
- 이자율을 바꿔보면 10년 후 금액이 어떻게 변하나요?

## 💡 생각해볼 점

- `=A1+B1`과 `=A1&B1`의 차이는 무엇일까요? (`&`는 문자 연결!)
- 수식에서 0으로 나누면(`=A1/0`) 어떤 일이 생길까요?
- 빈 셀을 참조하는 수식은 어떤 결과를 줄까요?
- 수식이 아주 많은 파일은 왜 열 때 느릴 수 있을까요?

## 💬 생각해볼 점 — 해답

**Q. `=A1+B1`과 `=A1&B1`의 차이는 무엇일까요?**
> - `=A1+B1`: **수학적 덧셈**. A1=10, B1=20이면 결과는 **30**
> - `=A1&B1`: **문자 연결**. A1=10, B1=20이면 결과는 **"1020"** (문자 그대로 붙임)
> `&`는 주로 이름과 성을 합치거나, 텍스트 조합 시 사용합니다.

**Q. 수식에서 0으로 나누면(`=A1/0`) 어떤 일이 생길까요?**
> **#DIV/0!** 오류가 발생합니다. 수학에서 0으로 나누기는 불가능하고, 엑셀도 마찬가지입니다. IFERROR로 감싸면 오류 대신 원하는 값을 표시할 수 있습니다.

**Q. 빈 셀을 참조하는 수식은 어떤 결과를 줄까요?**
> **0으로 처리**됩니다. `=A1+B1`에서 B1이 비어있으면 `=A1+0`과 같습니다. 다만, `=A1&B1`에서 B1이 비어있으면 빈 문자열(`""`)처럼 동작합니다.

**Q. 수식이 아주 많은 파일은 왜 열 때 느릴 수 있을까요?**
> 엑셀은 파일을 열 때 **모든 수식을 한 번에 재계산**하기 때문입니다. 수식이 수만 개면 계산 시간이 수십 초 걸릴 수 있습니다. 해결책: 수식 탭 → **수동 계산**으로 변경 후 필요할 때만 F9로 재계산.
